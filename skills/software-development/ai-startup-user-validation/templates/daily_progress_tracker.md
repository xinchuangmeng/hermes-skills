## 自动计算公式

### 关键指标公式
```
回复率 = 回复数 / 发送数
预约转化率 = 预约数 / 回复数
总体成功率 = 预约数 / 发送数
质量评分平均 = AVERAGE(质量评分列)
```

### 条件格式设置
1. **发送状态**：
   - ✅ 已发送：绿色
   - 🔄 跟进中：黄色
   - ❌ 未回复/已拒绝：红色

2. **质量评分**：
   - 9-10分：深绿色
   - 7-8分：浅绿色
   - 5-6分：黄色
   - <5分：红色

3. **回复时间**：
   - <2小时：绿色（积极）
   - 2-24小时：黄色（正常）
   - >24小时：红色（需跟进）

## 使用建议

### 每日操作流程
1. **上午9:00**：打开表格，更新昨日数据
2. **发送前**：填写目标信息，计算质量评分
3. **发送后**：立即更新发送状态和时间
4. **收到回复**：实时更新回复内容和时间
5. **下午17:00**：进行每日复盘，更新Sheet5
6. **晚上21:00**：检查未回复，准备次日跟进

### 数据备份
- 每日结束时导出为CSV备份
- 每周汇总生成周报
- 重要数据点截图保存
