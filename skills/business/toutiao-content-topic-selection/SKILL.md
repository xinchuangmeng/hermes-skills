# 今日头条内容选题技巧完整体系

## 一、热点追更方法

### 1.1 5大热点源

| 渠道 | 特点 | 用法 |
|------|------|------|
| 今日头条热榜 | 头条自家流量池，推荐加权最高 | 每天早中晚各刷一次 |
| 微博热搜 | 时效性最快 | 关注"实时上升"标签 |
| 百度风云榜/微信搜一搜 | 真实搜索量级 | 搜索量>50万算大热点 |
| 抖音热点榜 | 字节系跨平台联动 | 娱乐/生活热点首发多在抖音 |
| 行业垂直平台 | 精准流量 | 跨境看AMZ123/雨果网；AI看36氪 |

### 1.2 追热点3策略

**策略一：延迟追热**
大热点爆发前2小时是红海，6-12小时后反而更容易出彩
别人写"是什么"，你写"为什么"和"怎么办"

**策略二：冷门细分切入**
大热点找"没人写的角落"
示例：Temu进军南非 → 别人写"Temu又扩张了" → 你写"南非消费者对什么中国商品最买账"

**策略三：借势不蹭势**
热点做开头钩子，主体回归你的专业领域
公式：热点事件 + 领域关联 + 专业解读

### 1.3 同主题异角度切入法

| 角度 | 说明 | 跨境电商AI案例 |
|------|------|----------------|
| 技术层 | 怎么实现的 | "ChatGPT新模型凭什么这么强" |
| 应用层 | 怎么用 | "我用AI写listing转化率提升40%" |
| 商业层 | 赚钱吗 | "AI视频工具正在淘汰一类跨境运营" |
| 反思层 | 有什么坑 | "靠AI批量做内容3个月后账号废了" |
| 对比层 | 选谁更好 | "Claude vs ChatGPT 跨境写作选哪个" |

## 三、用户痛点挖掘

### 3.1 5大来源

1. **评论区挖金** — 自己/竞品爆文评论区收集高频问题
2. **搜索框挖掘** — 头条搜索下拉联想/百度相关搜索
3. **竞品分析** — 关注5-10个同领域头条号爆文选题
4. **行业论坛** — AMZ123/雨果网/Reddit/知乎
5. **自身经历+用户私信** — 自己做跨境遇到的困难就是选题

### 3.2 痛点转选题公式

```
原始痛点 → 场景化 → 标题化

示例：
"不会写英文listing"
→ "每次写亚马逊描述都头疼，翻译出来不像人话"
→ "3个Prompt让AI写出地道listing，转化率翻倍"
```

### 3.3 跨境电商核心痛点地图

| 痛点分类 | 具体痛点 | 选题方向 |
|----------|----------|----------|
| 选品类 | 不知道卖什么、怕选错 | AI选品方法 |

## 四、10大标题公式（附跨境电商AI案例）

| # | 公式 | 示例 |
|---|------|------|
| 1 | **数字定锚**：数字+领域+结果 | "3个AI工具让我每天少花2小时做运营" |
| 2 | **悬念钩子**：说结果不说原因 | "把Temu的9.9包邮品搬到亚马逊，结果..." |
| 3 | **反常识**：打破常规认知 | "别再学亚马逊运营了，不会AI的人正在被淘汰" |
| 4 | **痛点提问**：痛点+暗示方案 | "listing流量只来不转化？犯了致命错误" |
| 5 | **身份代入**：标签+场景 | "月销不到5万的小卖家，停止这3个操作" |
| 6 | **利益承诺**：好处+方法 | "0成本！用AI把视频效率提升10倍" |
| 7 | **对比反差**：A vs B | "同样的产品，别人100单你1单，差在这3处" |
| 8 | **紧急通知**：紧急+事件+行动 | "紧急！亚马逊新规，这类listing将被下架" |
| 9 | **内部揭秘**：揭秘+主题 | "大佬不愿说的5个选品秘密，全盘托出" |
| 10 | **故事冲突**：人物+冲突+结果 | "花10万买的教训：AI选品工具不是万能的" |

**标题优化技巧：** 双标题A/B测试、关键词前置前15字、情绪词(惊了/绝了/别再等)、短标题优于长标题(<25字)

## 五、选题评估

### 5.1 四维评估模型（1-5分/项，总分≥16值得写）

| 维度 | 评估问题 |
|------|----------|
| 热度 | 这个话题关注的人多吗？ |
| 点击潜力 | 标题能吸引人点开吗？ |
| 价值密度 | 内容能提供什么价值？ |
| 创作难度 | 我能写出差异化吗？ |

### 5.2 快速5问法

任何一个回答"否"就放弃或优化：
1. 别人会点开看吗？
2. 用户看完觉得有用/有共鸣吗？
3. 我能写出跟别人不一样的东西吗？
4. 有人会收藏/转发吗？
5. 跟账号定位一致吗？

## 六、选题库搭建方法

### 6.1 三级选题池

```
一级池(灵感池) → 二级池(待写池) → 三级池(排期池)
不限量            10-15个            7天排期
随手记录          评估后放入         按优先级发布
```

### 6.2 每日选题SOP（30分钟）

1. 刷头条热榜+微博热搜 → 记3-5个热点（5分钟）
2. 看3个竞品昨天发了什么 → 记爆文选题（5分钟）
3. 翻评论区收集用户问题（5分钟）
4. 搜行业关键词趋势（10分钟）
5. 整理到选题库标注优先级（5分钟）

### 6.3 内容发布节奏建议

- 周一/三/五：干货长文（头部内容）
- 周二/四/六：微头条短内容（快速试选题）
- 周日：复盘+选题规划
