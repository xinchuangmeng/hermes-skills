# AI智能体+跨境电商 短视频内容知识图谱构建

## 适用场景
- 准备做一个「AI智能体+跨境电商」交叉定位的短视频账号（抖音/视频号）
- 需要系统化了解赛道现状、爆款规律、学习路线
- 后期打算卖学习材料，需要提前储备知识体系
- 边做边学，需要明确的学习路径

## 完整工作流

### 第一步：全网搜索覆盖（8轮搜索）

并行搜索以下6个方向，每轮限5个结果：

**方向A：垂直赛道爆款规律**
- 搜索词1：`AI智能体 OR AI Agent 爆款视频 抖音 套路 2026`
- 搜索词2：`跨境电商 短视频 爆款 内容 账号 拆解`
- 搜索词3：`视频号 跨境电商 AI 爆款 账号 内容策略`

**方向B：标杆账号定位**
- 搜索词4：`抖音 AI智能体 跨境电商 万粉账号 拆解 内容策略`
- 搜索词5：`OpenClaw OR Hermes Agent 电商 实操 教程 视频`
- 在抖音/视频号内搜 `OpenClaw`、`Hermes Agent`、`AI Agent`、`豆包大模型 ArkClaw` 看高播放视频

**方向C：AI Agent学习资料**
- 搜索词6：`AI Agent learning resources roadmap 2026 GitHub 教程`
- 搜索词7：`学习AI智能体 最好的资料库 课程 Coze Dify LangChain`
- 重点提取：GitHub仓库（ai-agents-from-zero等）、roadmap.sh、在线课程

**方向D：变现机会洞察**
- 搜索词8：`AI Agent 课程 课程 价格 市场 竞争 2026`
- 分析市场上已有课程的定价和内容差异

### 第二步：关键页面深度提取

对以下类型的关键页面用 `web_extract` 提取详细内容：
1. GitHub仓库README（获取完整大纲和技术栈）
2. 教程平台的课程大纲页面
3. 拆解爆款账号的详细文章

### 第四步：内容规划输出

基于知识图谱，制定双平台内容规划表：

```
视频号（深度干货60-90秒）→ 剪成30秒版发抖音
抖音（快节奏25-45秒）→ 评论区导流私域
双平台统一终点 → 微信私域 → 成交
```

按阶段划分：
- 第1-4周：信任建立期（个人IP+科普）
- 第5-8周：引流转化期（工具实操+案例）
- 第9周+：变现放量期（复盘+矩阵）

### 第五步：建立持续学习循环

```
每周搜集AI+跨境新动态 → 整理成笔记
↓
每期视频输出 = 一次学习笔记的提炼
↓
累积到一定量 → 打包成课程
```

### 重要发现（避坑指南）

1. **AI+跨境电商 = 交叉蓝海**，目前抖音/视频号上没有强竞争者，是窗口期
2. 视频号更适合**深度干货**，抖音更适合**快节奏引流**，两条线不可混用
3. GitHub上 `ai-agents-from-zero` 项目是2026年最系统的中文教程，适合做参考
4. 不要一上来就卖课，先发30条以上视频建立信任
5. 每条视频本身 = 未来课程的一小节课，拍摄时注意积累素材

## 实操补充：情报→内容→变现完整流水线

### 6.1 情报存储结构

情报采集后不能散落在飞书消息里，要结构化存储：

```
~/.hermes/knowledge/情报站/
├── 每日简报/          ← 每天的情报快讯
│   ├── 2026-05-06.md
│   └── ...
├── 选题库/            ← 从情报中提取的可做视频的选题
│   ├── 待完成/
│   └── 已完成/        ← 发布后归档，附数据
├── 素材库/            ← 截图/数据/案例
└── 变现内容/          ← 整理后可卖的内容
    ├── 付费文章/
    ├── 教程/
    └── 工具推荐/
```

### 6.2 文案结构模板（1分钟口播）

```
【开头3秒】钩子——悬念/反常识/热点/痛点
  示例："泰国站费率涨了，说一个别人没注意到的……

【中间40-50秒】实操干货
  核心逻辑：为什么→怎么做→三步走
  用具体数字和操作步骤，不空谈概念

【结尾10秒】明天预告型钩子（前期适用）
  示例："明天我具体讲怎么算这个利润模型，关注我别错过"
  
注意：前期没有积累的资料时，用"明天分享XX"做钩子，
     不要用"评论XX发你资料"（因为没有资料可发）
```

### 6.3 视频发布规范

- 时长：50秒-1分10秒（初期），最长不超过1分30秒
- 视频号算法：完播率权重40% > 互动率30%
- 内容类型：实操干货型 > 热点解读型 > 故事型
- 全部AI生成（即梦/剪映），不做真人出镜
- 封面风格统一，字大醒目，色调一致

### 6.4 三平台联动策略

```
一源三发：一条情报 → 三种内容形式

视频号（每天1条）
  → 1分钟悬念口播短视频
  
公众号（每周2-3篇）
  → 同一话题的1500字长文 + 深度分析
  
头条号（每天1-2条）
  → 同一话题的200字微头条短评
```

## 产出物
- 爆款规律矩阵（可直接用于文案创作）
- 对标账号列表（用于日常拆解学习）
- 学习资料清单（用于自身学习 + 后期卖课素材）
- 双平台内容排期表（覆盖12周）

## Pitfalls
- 抖音上搜到的"爆款"可能是投流结果，不一定是自然流量
- 不下载/搬运别人视频，只做**模式分析**和**知识整理**
- 课程定价不要照搬市场价，你的差异化是"真实实操+个人IP"
- 搜索时Dify/Coze相关的内容偏中文生态，注意和跨境电商案例结合
