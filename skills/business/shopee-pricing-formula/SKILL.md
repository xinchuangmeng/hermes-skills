## 二、SLS跨境运费（从中国直发泰国）

| 项目 | 数值 | 说明 |
|:---:|:----:|:----|
| 计价方式 | **按实重** | 不计体积重 |
| 首重 | **10g起计** | 第一个10g |
| 续重 | **每10g** | 每个10g为一个计费单位 |
| 单价 | ≈**¥0.25/10g** | 约0.45฿/10g |
| 买家付运 | **≈¥10** | 统一价，不进卖家账户（不用管） |
| 时效 | 4-8天 | |
| 最大重量 | 通常≤**2kg** | 超重需走大件物流 |

```
SLS运费公式：SLS运费(¥) = (总克数 ÷ 10 - 1) × 0.25
```

> 注意：这里的SLS运费是针对标准跨境直邮（中国→泰国买家），不是海外仓发货的费用。

## 四、核心计算公式（强制统一算法）

### 【更新：仅作用于商品本身利润，不含藏价】
**藏价（SLS跨境物流成本）应单独计算并加入售价，详见运费计算器。**

### 第1步：SLS总运费
```
SLS总运费(¥) = (总克数 ÷ 10 - 1) × 0.25
```
> ⚠️ 此为估算值，实际费率以Shopee运费计算器（企业微信扫码下载Excel）为准。

### 第2步：真实总成本
```
总成本(¥) = 1688进价 + 国内运费 + SLS总运费（全额）
```

### 第3步：人民币售价
```
免佣期非活动：售价(¥) = 总成本 ÷ (1 - 0.02 - 目标利润率)
免佣期+活动：售价(¥) = 总成本 ÷ (1 - 0.07 - 目标利润率)
正常期非活动：售价(¥) = 总成本 ÷ (1 - 0.2063 - 目标利润率)
正常期+活动：售价(¥) = 总成本 ÷ (1 - 0.2563 - 目标利润率)
```

| 时期 | 完整公式 | 简化版 |
|:---:|:--------|:-----:|
| 免佣期非活动（目标利润25%） | 总成本÷(1-0.02-0.25) | **÷0.73** |
| 免佣期+活动（目标利润25%） | 总成本÷(1-0.07-0.25) | **÷0.68** |
| 正常期非活动（目标利润25%） | 总成本÷(1-0.2063-0.25) | **÷0.5437** |
| 正常期+活动（目标利润25%） | 总成本÷(1-0.2563-0.25) | **÷0.7437** |

### 第4步：泰铢售价
```
售价(฿) = 售价(¥) × 5
```

### 第5步：凑整标价
```
规则：按泰国人的心理价位凑整
¥范围  → 泰铢 → 标价
<¥10    → <฿50   → ฿39/49/59
¥10-20   → ฿50-100 → ฿69/79/89/99
¥20-50   → ฿100-250 → ฿129/149/179/199
¥50-100  → ฿250-500 → ฿249/299/399
```

### 第6步：利润验证
```
平台扣费 = 售价(¥) × 总费率
单笔利润(¥) = 售价(¥) - 1688进价 - 国内运费 - SLS总运费 - 平台扣费
利润率 = 单笔利润 ÷ 售价(¥)
利润率应 ≈ 目标利润率
```

**结论（市场调研+真实数据）：**

## 七、警告与约束（每次计算前必读）

1. ✅ **必须先确认1688包邮还是不包邮——不包邮要加国内运费（通常¥3~5）**
2. ✅ **必须先确认当前是免佣期还是正常期，以及是否参加免运活动**
3. ✅ **严格按公式顺序：先藏价→再总成本→再售价→再凑整**
4. ✅ **藏价用Shopee运费计算器查，宁可多藏不要少藏**
5. ✅ **买家含税总价（×1.07）用来跟竞品比价**
6. ✅ **正常期（非活动）目标利润率建议≥25%，否则扣除20.63%费率后利润薄**
7. ✅ **超过500g的商品，先问敬哥要不要做（利润可能太低）**
8. 🆕 **每次计算前先去虾皮大学确认当前最新费率！**
