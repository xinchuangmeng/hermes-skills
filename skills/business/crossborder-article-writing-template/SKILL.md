# 头条+公众号文章写作完整模板

## 一、头条版 vs 公众号版 核心差异

| 维度 | 今日头条 | 公众号 |
|------|---------|--------|
| **字数** | 500-800字+3-8图 | 1500-3000字深度 |
| **用户** | 三线以下为主，文化程度不高，易情绪化 | 一二线为主，有耐心，要深度 |
| **标题** | 冲突感+情绪+数字，标题党可适当 | 专业+价值+关键词，SEO友好 |
| **正文** | 短平快，分段密，每段2-3句 | 深度拆解，逻辑严密，引数据案例 |
| **图片** | 3-8张，配表情包/截图为佳 | 2-3张配图即可，重文字质量 |
| **引流** | 不能放微信，引导私信 | 可以直接放微信号/企微 |
| **推荐** | 算法推荐，标题决定打开率 | 社交推荐+搜一搜SEO |
| **收益** | 广告分成（完读率高才有收益） | 无广告分成，但私域转化价值更高 |

## 二、标题公式（通用，但头条更偏情绪类）

### 2.1 11种标题套路（按效果排序）

| # | 套路 | 公式 | 示例（跨境电商版） |
|---|------|------|-------------------|
| 1 | **数字式** | 数字+反差/结果 | 《开店第1个月，我只做这5件事，日销3000+》 |
| 2 | **冲突对比** | A对比B，制造反常识 | 《同行月销3000+？他靠的竟然不是投流，而是3步扒数据》 |
| 3 | **直击痛点** | 痛点+解决方案 | 《90%新手跨境开店都亏在这3个坑｜附避坑指南》 |
| 4 | **悬念式** | 留白，勾起好奇 | 《做了3个月跨境，终于发现为什么有人轻松月入5万...》 |
| 5 | **符号类** | 感叹号增强情绪 | 《太真实！新手做跨境电商，这3件事千万别碰！》 |
| 6 | **抱大腿** | 借势名人/热点/知名品牌 | 《跟Temu学选品：这个方法让我月销翻了3倍》 |
| 7 | **疑问式** | 提出问题 | 《做跨境电商，到底要不要先备货？我的答案是...》 |
| 8 | **人称代入** | "你""我"视角 | 《我用AI做跨境电商，1个人顶了3个人的活》 |
| 9 | **强调式** | 必须/千万不要 | 《做跨境电商，千万不要忽略这个细节》 |
| 10 | **贴标签** | 让人对号入座 | 《没资源没背景的人，怎么用AI做跨境电商？》 |
| 11 | **故事场景** | 细节场景引发共鸣 | 《月薪5千的我，辞职做跨境电商，3个月后...》 |

### 2.2 标题写作三步法

```
第1步：提炼文章核心价值（1句话）
  ↓
第2步：套上述11个公式中最适合的2-3个
  ↓
第3步：选择最吸引人的版本，发头条用情绪向，发公众号用专业向
```

**注意事项：**
- 标题前23字最关键（头条推荐展示的前23字）
- 头条版可"标题党"但不要文题不符
- 公众号版标题是搜索入口，要埋关键词

## 三、文章结构框架

### 3.1 头条版结构（500-800字）

```
【标题】 → 冲突/数字/情绪向
  │
【开头】 → 直接点题（3-5句内进入正文）
  │      → 公式：痛点引出/直接抛结论/场景代入
  │
【正文】 → 2-4个小标题，每个小标题200-300字
  │      → 每段2-4句，段落之间空行
  │      → 配图：每个小标题配1-2张图
  │
【结尾】 → 总结核心观点
  │      → 引导互动（"你觉得呢？评论区聊聊"）
```

### 3.2 公众号版结构（1500-3000字）

```
【标题】 → 专业+价值+关键词
  │
【引言】 → 故事/数据/场景切入，建立信任
  │      → 提纲挈领："今天分享3个方法..."
  │
【正文】 → 3-5个分论点，逻辑递进
  │      → 每个分论点：观点+案例+数据+金句
  │      → 段与段之间有过渡
  │      → 配图：2-3张关键截图/图表
  │
【总结】 → 金句收尾，升华主题
  │
【引流】 → 引导加微信/关注
  │      → "如果你想系统学习xxx，可以加我微信xxx"
```

### 3.3 基本结构模板（通用）

```
开头（10%）→ 吸引注意，点明主题
 问题/痛点引入
 或 数据和现象引入
 或 反常识观点引入
  ↓
核心内容（50%）→ 分模块拆解
 模块1：观点或案例 + 分析
 模块2：观点或案例 + 分析
 模块3：观点或案例 + 分析
  ↓
实际应用（15%）→ 给出可操作的方法
  ↓
结尾（10%）→ 金句总结 + 引导互动/引流
```

**金句穿插：** 每个模块结尾放一句精辟金句，增强深度和可读性

## 四、开头4种写法

| 方式 | 公式 | 示例 |
|------|------|------|
| **痛点式** | "你是不是也..."让读者觉得在说自己 | "你是不是也报了各种课、看了无数教程，结果跨境店铺还是没出单？" |
| **数据式** | 用具体数字建立可信度 | "我做跨境电商3个月，拿到了第一个月销5万的成绩，分享几个关键动作。" |
| **故事式** | 用一个小故事/场景代入 | "去年5月，我辞了工作，一个人开始做跨境电商。第一个月，挂了零。" |
| **反常识式** | 提出反常识观点，勾起好奇 | "你以为做跨境电商最重要的是选品？错了。最重要的是..." |

## 五、结尾4种写法

| 方式 | 示例 |
|------|------|
| **金句型** | "种一棵树最好的时间是十年前，其次是现在。" |
| **总结型** | "以上3个方法，是我3个月实操下来的核心总结，希望对你有帮助。" |
| **互动型（头条专用）** | "你觉得这个方法靠谱吗？欢迎评论区聊聊。" |
| **引流型（公众号专用）** | "如果你也想用AI做跨境电商，可以加我微信xxx，我整理了一份资料包。" |

## 六、排版规范

### 6.1 头条排版

| 格式 | 规范 |
|------|------|
| **段落** | 每段2-4句，不要超过5句 |
| **空行** | 段落之间必须空行，方便滑动阅读 |
| **小标题** | 用 **加粗** 或数字标号，方便扫读 |
| **字数** | 500-800字，配3-8张图 |
| **图片** | 截图/表情包/配图，每200字配1张 |
| **首图** | 标题下方配一张吸引眼球的图 |

### 6.2 公众号排版

| 格式 | 规范 |
|------|------|
| **段落** | 每段3-5句 |
| **小标题** | 用序号+加粗（一、二、三） |
| **字体** | 正文14-15px，标题加粗 |
| **引用** | 金句/数据用引用样式（>） |
| **图片** | 2-3张关键截图，质量要高 |
| **两端缩进** | 16px缩进，阅读更舒服 |

## 七、配图规范

### 7.1 头条配图

| 图位 | 类型 | 要求 |
|------|------|------|
| **首图（标题下）** | 吸引眼球，暗示内容 | 清晰、有冲击力、不放文字 |
| **正文配图** | 截屏/示意图/表情包 | 每200-300字配1张 |
| **结尾图** | 不必须 | 如放，放引导关注的图 |

- 头条建议用**三图模式**（标题下展示3张缩略图），比单图点击率高
- 图片质量不要太差，尽量清晰

### 7.2 公众号配图

| 图位 | 类型 | 要求 |
|------|------|------|
| **封面** | 吸引点击 | 清晰、有重点元素 |
| **正文配图** | 相关截图/数据图表 | 2-3张，高质量 |
| **引流图** | 引导加微信 | 文末，底部 |

## 你的跨境电商主题内容模板

### 选题方向

你适合发这4类内容：

| 类型 | 特点 | 头条版 | 公众号版 |
|------|------|--------|---------|
| **经验分享类** | 我做了什么、结果如何 | 500字短版 | 2000字深度版 + 数据 |
| **干货教程类** | 具体怎么做，步骤清晰 | 700字+配图 | 2500字+截图 |
| **避坑指南类** | 别人踩过的坑，你别踩 | 600字+情绪向 | 2000字+案例 |
| **工具推荐类** | AI工具/选品工具推荐 | 500字+清单 | 1500字+对比 |

### 8.2 内容来源

你的所有短视频文案都是现成素材：

```
一条短视频文案（60秒，约300-400字）
  ↓
头条版 → 扩展到600-800字，加配图，换标题
  ↓
公众号版 → 扩展到2000-3000字，加数据案例，深度分析
  ↓
两篇文章都发出去，互相引流
```

### 8.3 发布节奏建议

```
周一：你拍视频，我提取文案
周二：我写头条版 + 公众号版
周三：你审稿/修改
周四：小强发布到头条 + 公众号
周五：观察数据（哪篇数据好，下周做那条的进阶版）
周末：数据复盘
```

## 九、避坑指南

### 头条避坑
1. **不要一篇内容不改全平台发** — 会被判低质降权
2. **标题不要太文艺** — 头条用户要直白
3. **不要太长** — 超过1000字完读率会暴跌
4. **不要忽略评论运营** — 发完去评论区引导互动
5. **内容要踩大标签** — 你的"跨境电商""AI"属于商业/科技类，标签不冷门

### 公众号避坑
1. **标题要埋搜索关键词** — 微信搜一搜流量越来越重要
2. **文章要深，不要浅** — 公众号用户要干货不要水货
3. **文末必须放钩子** — 个人微信号、企微随意放，这是公众号最大的引流优势
4. **保持更新频率** — 至少每周1篇
5. **重视评论区** — 回复评论建立信任
