# 微信公众号基础运营+数据复盘完整知识

## 一、订阅号vs服务号

| 维度 | 订阅号 | 服务号 |
|------|--------|--------|
| 推送频率 | 每天1次(8篇) | 每月4次 |
| 显示位置 | 折叠在订阅号文件夹 | 直接显示聊天列表 |
| 注册主体 | 个人可注册 | 仅限企业(需营业执照) |
| 认证费用 | 可选(300元/年) | 必须认证(300元/年) |
| 高级接口 | 基础 | 微信支付/模板消息/网页授权等 |

**结论：敬哥做内容IP，选订阅号。**

## 二、菜单栏规划

一级菜单最多3个，每个最多5个二级菜单，不支持三级。

推荐配置(敬哥版)：
```
AI短视频      跨境干货      联系我
 ├ 教程        ├ 选品技巧     ├ 商务合作
 ├ 脚本模板    ├ 工具推荐     ├ 加入社群
 └ 案例拆解    └ 资源下载     └ 关于我
```

菜单三种响应：跳转网页/小程序/点击推事件(自动回复)

## 三、自动回复

### 被关注回复
欢迎语+定位介绍+引导动作(回复关键词/点菜单)
控制在150字以内，给出明确下一步指引

### 关键词回复
- 每条规则最多10个关键词，用回车分隔
- 全匹配：必须完全一致才触发
- 半匹配(推荐)：包含关键词就触发
- 回复全部：一次性发送所有内容
- 随机回复：只随机选一条发送

### 消息回复(兜底)
用户发消息没命中任何关键词时触发，引导用关键词或菜单

## 四、数据复盘方法

### 公众号核心指标

| 指标 | 基准值 |
|------|--------|
| 打开率(会话阅读率) | >1%合格，>2%优秀，>5%爆款 |
| 分享率 | 越高越好 |
| 留言率 | 越高越好 |
| 关注转化率 | 新增关注÷文章阅读人数 |
| 常读用户比例 | 公众号质量核心指标 |

### 头条号核心指标

| 指标 | 基准值 |
|------|--------|
| 点击率(CTR) | <1.5-2%停止推荐，>5%优秀 |
| 完读率 | 越高越好 |
| 互动率 | 评论>转发>点赞 |
| 涨粉率 | 新增关注÷阅读量 |

### 复盘方法论

1. **四象限法**：打开率(横轴)×分享率(纵轴)分四类
2. **漏斗法**：展示→点击→互动→关注→转化逐层分析
3. **聚类法**：按选题/时间/形式聚类分析
4. **时间序列**：按日/周/月追踪趋势

### 复盘频率
- 日：核心数据速览(5分钟)
- 周：文章表现+互动+粉丝(30分钟)
- 月：内容类型效果+平台表现(1小时)
