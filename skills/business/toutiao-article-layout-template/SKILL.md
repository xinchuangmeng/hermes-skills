# 今日头条图文排版完整模板

## 编辑区参数

| 参数 | 值 |
|------|-----|
| 正文字号 | 16px |
| 行间距 | 1.75倍 |
| 段间距 | 空一行 |
| 每段行数 | 3-5行 |
| 字间距 | 1px |
| 对齐 | 两端对齐 |
| 首行缩进 | 不需要 |

## 标题与正文层级

| 层级 | 字号 | 颜色 | 格式 |
|------|------|------|------|
| 小标题 | 18-20px | 加粗深色#333 | 统一序号(一/二/三) |
| 正文 | 16px | #505050 | 两端对齐 |
| 金句/重点 | 16px | 加粗 | 单独成段 |
| 注释 | 13-14px | #999 | 图片下方 |

## 配图规范

- 首图：900×383px (2.35:1)，核心信息放中央1:1安全区
- 三图模式：每张200×200px，风格统一
- 正文插图：每500-800字1张，前后空行
- 禁用：二维码/微信号/水印/低质图

## 文章结构

```
标题(在头条标题区写，正文不重复)

直奔主题(2-3句切入痛点/抛观点)

一、小标题(加粗)
 2-3段正文 (每段3-5行，段间空行)
 [配图]

二、小标题
 2-3段正文
 [配图]

三、小标题
 2-3段正文

结尾：引导互动
 "你觉得呢？评论区聊聊"
```
