# 公众号推文标准结构+双平台标题模板

## 一、推文标准结构

### 开头(10-20%)——7种钩子

1. 描述痛点："做跨境3个月不出单？90%的新手都踩过这3个坑"
2. 制造悬念："我花5000买来的教训..."
3. 讲个故事："去年我还是个跨境小白，直到..."
4. 引入热点："TikTok Shop全托管上线了，中小卖家还有机会吗？"
5. 框架前置："本文从选品→拍摄→投流，一次性讲透"
6. 金句/数据："2025年跨境短视频转化率平均提升37%"
7. 开门见山："直接说结论：..."

### 中段(60-70%)——SCQA结构

01 背景/问题(300-500字)
02 核心干货2-5个点(每点300-800字，配案例+截图+数据)
03 案例/实战拆解(300-500字)
04 总结/方法论模型(200-300字)

### 结尾(10-20%)——7种方式

1. 总结全文+核心观点回顾
2. 金句升华(引发共鸣与转发)
3. 话题互动("你觉得呢？评论区聊聊")
4. 下期预告(促关注)
5. 抛诱饵/资料包("后台回复XX领取")
6. 引导三连(点赞/在看/转发)
7. 固定Slogan(建立IP记忆点)

结尾必须包含明确的CTA(关注/点赞/评论/加微信)，1-3个

## 二、排版配色规范

| 规范项 | 值 |
|--------|-----|
| 正文字号 | 15-16px |
| 行间距 | 1.75-2.0倍 |
| 段间距 | 空1行 |
| 字间距 | 0.5-1px |
| 页边距 | 左右10-15px |
| 小标题 | 16-18px，加粗，品牌色 |
| 正文色 | #333333~#555555(不用纯黑) |
| 主色 | 1个品牌色(小标题/强调/链接) |
| 辅助色 | 1个高亮色 |
| 全文颜色 | ≤3种(主色+辅色+黑白灰) |
| 引用/案例 | 浅灰色背景区块或左侧竖杠 |
| 图片 | 宽度640px以内，横版16:9 |
| 对齐 | 正文两端对齐，标题居中或左对齐 |

## 三、12大通用标题公式(跨境电商AI版)

| # | 公式 | 示例 |
|---|------|------|
| 1 | 数字+方法+结果 | 3个AI工具让视频效率提升10倍 |
| 2 | 疑问+痛点 | 为什么你拍的视频总是没转化？ |
| 3 | "我"字故事+转折 | 我测了50款AI工具，只留了这3个 |
| 4 | 否定式+利益 | 别再手动翻译了！AI一键生成多语言视频 |
| 5 | "原来"+认知颠覆 | 原来短视频爆量的秘密不在于拍摄技术 |
| 6 | "我劝你别"+反向吸引 | 我劝你别轻易做跨境AI视频，除非看完这篇 |
| 7 | XX天+结果+方法 | 30天从0到月销50K，只做对了一件事 |
| 8 | 对比+数据 | 同样产品他播放100万你只有200？差在哪 |
| 9 | 热点+观点 | TikTok全托管新规来了，这4类卖家要小心 |
| 10 | 揭秘/干货+承诺 | 保姆级教程：用AI搞定短视频全流程 |
| 11 | "过来人"+教训 | 跨境老司机翻车总结：AI视频5个坑 |
| 12 | 清单体+实用 | 跨境卖家必备8个AI神器(建议收藏) |

## 四、头条vs公众号标题适配

| 维度 | 公众号 | 头条号 |
|------|--------|--------|
| 风格 | 理性/深度/实用 | 情绪化/直接/接地气 |
| 字数 | 16-20字 | 前23字最关键 |
| 关键词 | 核心词前5字(搜一搜SEO) | 前15字内 |
| 情绪词 | 适度用 | 可以多用(惊了/绝了/别再) |
