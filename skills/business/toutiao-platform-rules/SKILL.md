# 今日头条平台规则完整知识体系

## 一、推荐机制

### 1.1 核心原理：分级推荐（赛马机制）

内容不依赖粉丝基础，通过算法逐级筛选，数据好的持续获得更大流量。

**流量池递进层级：**

| 层级 | 曝光量级 | 触发条件 |
|------|---------|---------|
| 冷启动池 | 200~1000人 | 审核通过自动进入 |
| 第1级流量池 | 3,000~5,000人 | 冷启动数据达标 |
| 第2级流量池 | 1万~10万人 | 上一级达标 |
| 第3级流量池 | 10万~100万人 | 持续优秀 |
| 爆发池 | 100万~1000万+ | 成为爆款 |

**关键时间窗口：** 80%以上推荐量在发布后24小时内完成，理论推荐周期48-72小时。

### 1.2 冷启动机制

- 审核通过后系统自动分配200-1000次初始曝光
- 推荐对象：算法根据内容标签匹配用户群
- 2025年起头条对深度文章设有专门冷启动流量池

### 1.3 核心评估指标

| 指标 | 权重 | 说明 |
|------|------|------|
| 点击率(CTR) | ★★★★★ | 标题+封面决定。<1.5-2%停止推荐，>5%优秀 |
| 完读率/阅读时长 | ★★★★★ | 短文看完成率，长文看停留时长 |
| 互动率 | ★★★★ | 评论>转发>点赞，评论质量影响二次推荐 |
| 收藏率 | ★★★ | 代表长期价值，有助于长尾推荐 |
| 跳出率 | ★★★ | 快速跳出会降低权重 |

### 1.4 头条号指数（账号权重）

**5个维度：**
- 健康度：合规性，无违规记录
- 关注度：粉丝增长和活跃
- 传播度：被转发分享的广度
- 垂直度：内容领域专注度
- 原创度：原创内容比例（最重要的加分项）

**正向循环：** 头条号指数高 → 初始推荐量大 → 数据好 → 指数继续上升

## 二、流量分发逻辑

### 2.1 标签匹配

- 内容标签：系统自动解析关键词/主题/领域
- 用户标签：基于历史行为构建的兴趣画像
- 匹配：内容标签 × 用户标签 → 最可能喜欢的用户

### 2.2 同主题去重

- 两篇文章连续13个词完全一致，或全篇60%以上重合 → 判定重复
- 原创度<70%：可能被判定低质内容
- 原创度<60%：有抄袭风险，可能封号

### 2.3 权重因素

- 账号权重高 → 初始推荐量大
- 首发+原创 → 额外加权
- 热点内容优先推荐，过时内容(>3天)大幅降权
- 粉丝触达有特殊加权

## 三、违规红线

### 3.1 敏感内容（扣70分/直接封禁）
违反法律法规、诱导低俗、谣言不实内容

### 3.2 营销硬广（扣20分）
- 借领导人/重大会议营销（绝对红线）
- 推荐风险商品（传销/网贷/荐股/违禁品）
- 内容无信息增量（纯商品参数无专业内容）
- 诱导私信引流（加微信/留联系方式）
- 使用极限词（"国家级""最XX""第一"）

### 3.3 搬运/洗稿判定
- 抄袭：扣10分+下架
- 洗稿（复制拼凑/换汤不换药）：扣20分+下架
- 搬运（直接复制）：扣10-70分+封号风险

### 3.4 标题党（扣20分）
夸张式（"震惊/重磅"）、强迫式（"不看后悔"）、歧义式、无中生有

### 3.5 其他违规
- 过时效内容（旧闻新发）：扣10分
- 违规互粉：扣20分
- 格式低质（乱码/机翻/无标点）：扣10分
- 违规声明头条首发：扣5分

## 四、信用分体系

| 分值 | 权益状态 |
|------|---------|
| 100分（初始满分） | 全部权益正常 |
| 61-99分 | 可使用已有权益 |
| ≤60分 | **全部权益关闭！** |
| ≤30分 | 禁言7天 |
| 0分 | **永久封禁** |

**恢复方式：** 每连续10天无违规恢复10分；首次违规可答题恢复

## 五、发稿注意事项

| 项目 | 规则 |
|------|------|
| **头条首发条件** | 内容首先在头条发布，72小时内未发其他平台 |
| **首发违规** | 声明首发但不符合条件 → 扣5分 |
| **最佳发布时间** | 早7-9点、午11:30-13:30、晚19-23点 |
| **每日发文量** | 建议1-3篇/天，至少3-5篇/周 |
| **忌** | 连续大量发、密集发、多天不发芽然爆发 |

## 六、对敬哥的具体建议

### ✅ 推荐内容方向
1. 跨境电商干货（高收藏率）
2. AI工具测评（高互动率）
3. 行业趋势解读（深度内容）
4. 案例复盘

### ✅ 规避红线措施
1. 不要直接推荐具体产品，用"方法"替代"产品"
2. AI生成内容要二次加工（加个人观点和经验）
3. 不做直接营销（"今天教你XXX，点击购买"）
4. 不用极限词（"第一/最/100%"）
5. 文章优先在头条发布，72小时后再同步到公众号
6. 每天固定发文时间（建议早8点或晚18点）
