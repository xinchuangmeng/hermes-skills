### 1.2 TikTok流量分发底层逻辑

#### 1.2.1 核心机制：流量池逐级筛选
```
发布视频 
  → 第1级流量池（200-500播放量：冷启动测试）
    → 完播率>30% + 互动率>5% → 第2级（2000-5000）
      → 完播率>35% + 分享率高 → 第3级（1万-5万）
        → 持续正向数据 → 百万级爆款
        → 数据不达标 → 停止推流（死在当前池）
```

**核心指标权重排序：**
1. **完播率**（最重要）—— 前3秒留住用户决定一切
2. **互动率**（点赞/评论/转发/收藏）—— 告诉平台内容有价值
3. **分享率**（分享到站外）—— 破圈核心指标
