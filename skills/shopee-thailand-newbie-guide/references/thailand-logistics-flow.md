# 客户购买路径与付款流程（泰国站）

> 从海外仓/一件代发到泰国买家手中的完整链路

## 场景一：中国直发（国际小包）
```
1688供应商 → 国内中转仓贴面单 → 国际小包（3-7天）→ 泰国海关 → 泰国买家
运费：按g计费，首重¥10+续重每10g¥0.25
```

## 场景二：海外仓发货（大件品适用）
```
1688批量采购 → 海运拼柜（20-30天）→ 泰国海外仓 → 泰国本地物流 → 泰国买家
运费：海运分摊¥5-10/个 + 本地配送¥6-10
利润更高，但需提前备货
```

## 付款流程
1. 泰国买家在Shopee下单付款（泰铢）
2. Shopee冻结款项（买家确认收货后释放）
3. 卖家发货（直接发或通知海外仓发）
4. 买家收货确认
5. Shopee打款给卖家（泰铢→卖家后台）
6. 卖家提现到银行卡

## 退货处理
- 跨境退货≈全亏（退货运费>商品价值）
- 理由：买家退货→商品退回中国运费¥50+ → 等于报废
- 对策：选低售后品类（收纳/五金/硅胶/地垫）
- 对策：泰国海外仓支持本地退货再销售
