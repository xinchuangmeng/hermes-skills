## 一、佣金、服务费及交易手续费（2026年版）

### 1.1 费率总表

| 费用类型 | 费率 | 说明 |
|:--------|:----|:-----|
| 平台佣金（泰国站SLS） | **18.63%** | 新卖家前3个月免佣（每月限500单） |
| 交易手续费 | **2%** | 所有站点统一，支付渠道费 |
| 服务费（FSS免运等） | **5%** | 仅参加免运/平台活动时收取 |
| 合计（非活动期，免佣期） | **2%** | 只交交易手续费 |
| 合计（非活动期，正常期） | **20.63%** | 佣金18.63%+交易费2% |
| 合计（活动期，正常期） | **25.63%** | 佣金18.63%+交易费2%+服务费5% |

### 1.2 免佣政策

- **生效时间：** 2026年1月1日起
- **适用站点：** 新加坡、马来西亚、**泰国**、越南、菲律宾、台湾
- **免佣期限：** 成功开通的每个店铺，自动免除**前3个月**佣金
- **上限：** 每月免佣订单数量上限为**500单**
- **拓站计划：** 对新开店铺还提供佣金减免、广告金激励、资源支持

### 1.3 佣金计算公式

```
佣金 = (商品原价 + Shopee提供的商品补贴 - 买家优惠折扣) × 佣金费率
```

### 1.4 交易手续费计算公式（泰国站）

```
交易手续费 = 订单总付款(使用优惠券/Shopee币后，含买家支付运费的全额) × 2%
```

### 1.5 服务费说明

服务费涵盖：
- 平台功能使用
- 卖家计划/活动参与资格（如FSS/CCB）
- 数字基础设施优化（平台基础设施费）

**注意：** FSS/CCB服务费已并入平台基础服务，但**仅参加活动的商品才扣**。不参加活动不扣。

## 二、跨境物流成本（藏价）与SLS运费

### 2.1 藏价是什么

```
卖家支付运费 = 跨境物流成本(藏价) + 买家支付运费
                      ↓
             你需要加入商品价格的部分
```

**关键：**
- **藏价 = 你实际承担的物流成本**
- 买家支付运费 = 买家付的（或平台补贴的）
- **藏价必须加入商品售价**，不管是否包邮

### 2.2 SLS渠道分类

| 渠道类型 | 说明 |
|:--------|:-----|
| 普货 | 普通SLS渠道，大多数卖家适用 |
| 特货 | 特殊商品渠道（具体分类见物流手册） |
| 重货 | 大件/重货渠道 |

### 2.3 藏价多少合适

**原则：宁可多藏，不要少藏**
- **少藏：** 会亏运费（最差情况）
- **多藏：** 只是价格竞争力弱一点（可通过定价/活动平衡）

**调整幅度建议：** 每件商品藏价可调整±20%

### 2.4 多件出单的藏价策略

| 场景 | 建议 |
|:----|:-----|
| 单品独立出单（>70%） | 藏全部运费 |
| 多件组合出单（>70%） | 藏部分运费（如藏1/3~2/3） |
| 引流款/活动款 | 可少藏或不藏（以低价引流为目的） |

### 2.5 运费查看路径

```
中国卖家中心 → 财务 → 订单结算 → 查看订单详情
```

在该页面可以看到：
- **跨境物流成本（实际）**：最终你承担的运费（可能与你的估算有浮动）
- **买家支付运费**
- **卖家支付运费**

### 2.6 常见误区

| ❌ 误区 | ✅ 正解 |
|:-------|:--------|
| 藏价加入商品价格=包邮 | 藏价是你承担的物流成本，不管包不包邮都要藏入 |
| 重量估少了藏价补上就行 | 少藏会亏损，宁可多藏 |
| SPM/包邮计划也要自己藏价 | 不！SPM/包邮计划运费由Shopee承担，无需藏价 |

## 三、VAT税（泰国站）

请注意：本参考文件整理时OCR未提取到完整的泰国VAT专项政策页面。以下信息来自之前对话中确认的内容：

- **泰国VAT税率：7%**
- **承担方：** 买家承担
- **Shopee处理方式：** 系统自动在商品售价上加7%展示给买家
- **对卖家的影响：** 不进卖家收入，但影响买家竞价
