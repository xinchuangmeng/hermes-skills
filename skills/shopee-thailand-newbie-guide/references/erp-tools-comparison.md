## 推荐工作流

```
1688选品 → 妙手采集商品信息
         → 自动翻译泰语标题/描述
         → 自动生成Listing
         → 一键发布到Shopee泰国店
         ↓ (有人下单)
妙手自动去1688代发采购
         → 自动处理国际物流
         → 买家收到货
```
