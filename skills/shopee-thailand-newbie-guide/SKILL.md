## 一、入驻准备

### 所需材料
- 营业执照（个体户即可，经营范围含"电子商务"）
- 法人身份证
- 手机号（接收验证码）
- 邮箱（建议126/163邮箱）

### 入驻流程
1. 打开 https://seller.shopee.cn → 点立即入驻
2. 选择泰国站
3. 上传营业执照+法人身份证
4. 填写店铺信息（店名、品类、联系方式）
5. 提交审核 → 3-7个工作日通过

## 二、选品策略

### 选品三铁律
| 条件 | 标准 | 原因 |
|------|------|------|
| 轻小件 | <200g最佳 | 跨境运费按重量算，越轻利润越高 |
| 低售后 | 不用说明书 | 跨境退货≈报废 |
| 无侵权 | 无品牌logo/卡通形象 | 封店风险 |

### 大件品利润逻辑（本地仓/海运模式）

> **用户常见疑问：** "这种大件运费都不止卖价了，卖家怎么赚钱？"

大件品（如碗碟架、收纳柜）在Shopee泰国站卖得动，是因为卖家不是从中国一件代发，而是用以下模式：

| 模式 | 说明 | 利润率 |
|:----|:-----|:------|
| **海运到泰国海外仓** | 1688拿货¥10-20，海运拼柜到海外仓（¥5-10/个），泰国本地配送（¥6-10） | **40-50%** |
| **泰国本地货源** | 泰国本就有工厂/批发商直接在Shopee卖，无国际运费 | 灵活 |
| **新加坡集运仓** | 新加坡中转站批量收货后转发泰国，运费低于国际小包 | **35-45%** |

**关键：** 你在Shopee看到的大件低价品，大概率是**有海外仓或本地货源**的卖家。新手期不建议碰这些——退货损失大、资金压力大。

**新手正确路径：** 先用轻小件（<200g）跑通模式，稳定出单后再逐步上海运大件品。

### 选品方法
- 1688搜"跨境 一件代发"关键词
- 筛选条件：一件代发✓ 包邮✓ 48H发货✓
- 排序：先按销量排看爆款，再按价格排找供应商
- 使用选品数据工具：见 `references/erp-tools-comparison.md`（知虾/妙手ERP选型对比）

|:----:|:----------------:|:---------------:|:-----------:|------|
| 佣金 | **0%** | **18.63%** | 18.63% | 虾皮大学官方数据 |
| 交易手续费 | **2%** | **2%** | 2% | 所有站点统一，支付渠道费 |
| 服务费（FSS免运等） | 不参加不收费 | 不参加不收费 | **5%** | 仅参与免运/平台活动时才收 |

### 新手期定价（前3个月，核心算法）

新手期前3个月只有一笔平台费用：**交易手续费2%**。

```
总成本 = 1688进价 + 国内运费 + 藏价（SLS跨境物流成本）
售价(¥) = 总成本 ÷ (1 - 0.02 - 目标利润率)
       = 总成本 ÷ (0.98 - 利润率)
售价(฿) = 售价(¥) × 5 → 凑整
```

**藏价怎么算？**
- 藏价 = 你支付的SLS总运费 - 买家承担的运费（平台标准补贴）
- 使用Shopee运费计算器（企业微信扫码获取Excel）
- **宁可多藏不要少藏**，少了你自己贴钱
- 大致的参考：600g泰国商品，藏价约¥14.75（旧标准每10g¥0.25估算）
- 精确费率需查运费计算器Excel表

**实例：**
```
进价¥4.94 + 国内¥3 + 藏价¥14.75 = 总成本¥22.69
目标利润率25%：
售价(¥) = 22.69 ÷ (1 - 0.02 - 0.25) = 22.69 ÷ 0.73 = ¥31.08
售价(฿) = 31.08 × 5 = ฿155 → 凑整标฿159
利润验证：31.08 - 22.69 - (31.08×2%) = ¥7.77，利润率25% ✅
买家含税：฿159 × 1.07 = ฿170（用这个跟竞品比价）
```

### 两步定位法拓展版

> 新手期计算公式简化版：
> 售价(¥) = 总成本 ÷ (0.98 - 0.25) = 总成本 ÷ 0.73（目标利润25%时）
> 售价(฿) = 售价(¥) × 5
> 买家含税总价(฿) = 售价(฿) × 1.07
> 用含税总价跟竞品比价

### 多规格阶梯定价法（三级定价）

| 档位 | 利润率 | 目的 | 心理学 |
|:----:|:------:|:----:|:------:|
| **S/基础款** 🎣 引流价 | **5-15%** | 拉流量、拉权重 | "好便宜，先进来看" |
| **M/主推款** 💪 主力价 | **25-35%** | 出单主力 | "比S大好多，划算" → 70%选这个 |
| **L/利润款** 👑 利润价 | **40-60%** | 赚高利润 | 给用户对比的锚点 |

### 强制约束（新手期专用）
1. ✅ 必须先确认1688包邮还是不包邮——不包邮要加国内运费
2. ✅ 当前是免佣期，总费率确认**仅2%交易手续费**
3. ✅ 严格按公式顺序：先藏价→再总成本→再售价→再凑整
4. ✅ 藏价用Shopee运费计算器查，**宁可多藏不要少藏**
5. ✅ 最后必须给凑整建议价 + 买家含税价（跟竞品比）

## 四、低价引流策略

> 核心：引流款不靠自身赚钱，作用是拉权重+引流量+带动利润款

### 引流款选品标准
- 1688采购价 ¥1-5
- 重量 <100g
- 不碎不坏（硅胶/塑料）
- 刚需高频

### 店铺权重与SKU数量的误区

| ❌ 误区 | ✅ 真相 |
|:-------|:--------|
| 铺货越多流量越多 | 平台看转化率，1000个死链接不如50个稳定出单品 |
| 先铺500个再说 | 长期不出单的品拉低店铺整体转化率，反而降权 |
| 泰国站喜欢杂货铺 | 泰国买家更青睐**品类专一**的店，算法好给你贴标签 |

**正确做法：**
- **第一批上30~50个精选品** → 观察7天哪些出单
- **淘汰不出单的** → 换新10~20个
- **稳定出单的品做深** → 扩SKU（颜色/尺寸变体）
- **新品限额：** 新店泰国站约500个SKU上限，超出被系统自动隐藏

### 引流款的作用
| 作用 | 说明 |
|------|------|
| 拉店铺权重 | 低价款易出单，提升搜索排名 |
| 带动利润款 | 用户进店后逛到利润款下单 |
| 做动销率 | 避免店铺因无订单被降权 |
| 参加活动 | 低价款容易被选入平台活动 |
| 广告引流 | 获客成本极低，ROI高 |

## 六、店铺权重与排名

### 影响权重的因素
| 因素 | 说明 |
|------|------|
| 销量 | 订单越多权重越高 |
| 好评率 | 评价分影响排名 |
| 动销率 | 店铺需持续出单，无订单会被降权 |
| 上新频率 | 定期上新品获流量扶持 |
| 回复率 | 客服响应速度 |
| 发货速度 | 快速交付率影响权重 |

### 提升权重的操作
- 保持每天上新3-5个品
- 设置关注礼券引导粉丝积累
- 低价引流款保底出单
- 参加平台活动获取额外流量
- 优化标题关键词提升搜索排名
