# Shopee平台真人化浏览风控规避标准

## ⚠️ 重要前提
- 此技能专为**小强（OpenClaw，Windows本地）**或**旺财（Hermes微软版，Windows本地）**使用
- **小书童（Hermes，腾讯云服务器）**IP已被电商平台封禁，无法执行
- 此技能适用于纯学习查阅规则、手动采集规则数据场景，**禁止用于批量采集/铺货/刷单**

---

## 一、基础环境真人化

### 1.1 浏览器标识伪装
- 随机切换真实手机/电脑UA
- 固定常用浏览器型号（建议Chrome 120+或Edge 120+）
- 不使用机器人专属标识（HeadlessChrome、Puppeteer等标记）

### 1.2 窗口行为
- 不定时微调浏览器窗口大小（每次±10-30px变化）
- 不全屏固定尺寸
- 模拟人手拖动习惯

### 1.3 网络环境
- 固定常用地区IP
- 频繁切换地区IP会判定异常
- **严禁使用代理/VPN频繁切换区域**

---

## 二、浏览节奏真人化（核心防风控）

### 2.1 停留时长规则
| 页面类型 | 最小停留时间 | 说明 |
|:--------|:-----------|:----|
| 新打开的任意页面 | **8-15秒** | 禁止秒进秒退 |
| 规则/文档页 | **30秒+** | 模拟阅读时间 |
| 商品详情页 | **20秒+** | 模拟看商品 |
| 后台数据页 | **15-25秒** | 模拟查看数据 |

### 2.2 翻页速度
- 慢速滚动浏览内容
- 逐段阅读，每段停顿2-5秒
- **禁止一键到底快速拉到底**
- 使用模拟鼠标滚轮滚动（每次3-5行）

### 2.3 间隔休息
- 每操作 **3个页面**，自动停顿 **20-40秒**
- 静默休息，不做任何操作
- 模拟人起身喝水/看手机的习惯

### 2.4 随机闲逛
- 查完规则/数据后，**必须**随机点2-3个无关商品/页面闲逛
- 不直奔目标后立即退出
- 闲逛时也按正常速度浏览

---

## 三、点击操作真人化

### 3.1 点击位置
- 点击按钮**偏边缘一点**（偏移5-15像素）
- 不精准点击正中心坐标
- 使用随机偏移量

### 3.2 连续点击限制
- 同页面连续点击间隔 **≥3秒**
- 不连续快速点多个功能
- 每次点击间加随机延迟（3-5秒）

### 3.3 误触模拟
- 偶尔轻微点错空白位置（概率约10%）
- 再重新点击目标位置
- 模拟真人的"手滑"行为

---

## 四、账号登录与会话真人习惯

### 4.1 登录方式
- **优先模拟扫码登录**
- 少用密码自动填充登录
- 不使用Token/API密钥直接登录网页版

### 4.2 登录频率
| 时段 | 最大登录次数 | 说明 |
|:----|:-----------|:----|
| 一天内 | **2-3次** | 不反复上下线 |
| 1小时内 | **最多1次** | 登录间隔≥1小时 |

### 4.3 在线时长
- 单次在线浏览：**30分钟内自动离线休眠**
- 不24小时挂着
- 如需继续查看，重新登录

---

## 五、Shopee平台专属真人行为

### 5.1 浏览顺序
- 先看**店铺后台数据** → 再逛**前台买家页面**
- 顺序贴合卖家日常操作习惯

### 5.2 模拟日常打理
- 偶尔查看站内消息
- 查看系统通知
- 模拟正常打理店铺的操作流

### 5.3 规则阅读习惯
- 查看运费表、定价规则时：**来回上下反复看2遍**
- 不是一次性读完就走
- 模拟"确认一下"的真人习惯

### 5.4 禁止行为清单
| ❌ 禁止 | 原因 |
|:-------|:----|
| 批量采集商品数据 | 高频风控触发 |
| 批量导出 | 明显机器行为 |
| 批量一键铺货 | 敏感高风险操作 |
| 高速翻页（<3秒/页） | 非人行为 |
| 同时打开10+标签页 | 非人行为 |

### 5.5 退出习惯
- 查完规则后**主动退出部分子页面**
- 不长期停留在规则文档页
- 保留1-2个前台页面假装在看商品

```
小强/旺财（本地Win浏览器，加载本技能）→ 真人浏览截图
→ 飞书群/私聊发截图给敬哥
```

| 要求 | 说明 |
|:----|:----|
| 清晰度 | 截大图，不要裁边 |
| 表格页 | 确保百分比列完整可见 |
| PDF/Excel附件 | 截图内注明文件名，单独发一份 |
| 发送方式 | 飞书群/私聊，打上#虾皮大学等标签 |

## 九、虾皮大学专项规则（抓取官方源时的补充约束）

当执行「虾皮大学知识采集」任务时，额外遵守以下规范：

### 9.1 目标页面优先级
| 优先级 | 板块 | 说明 |
|:------|:----|:----|
| 1️⃣ | 新手入门 → 新手须知规则 | 佣金/服务费/交易费数据 |
| 2️⃣ | 账户与财务 → 费率表 | 29.47%总费率来源 |
| 3️⃣ | 泰国站专区 → 税务&关税&SLS | VAT 7%、关税、物流规则 |
| 4️⃣ | 物流 → SLS跨境物流手册 | 运费表、禁运品清单 |
| 5️⃣ | 店铺健康 → 惩罚积分 | 扣分/解封/季度重置规则 |

### 9.2 截图要求
- **每页截清晰大图，不要裁边**（避免漏掉关键信息）
- 如页面有展开/折叠内容，**全部展开后再截**
- 每张截图需能看清所有文字
- 截完后在群里打标签 **#虾皮大学**

### 9.3 与shopee-thailand-newbie-guide技能的配合
截完图后，截图发给小书童（云端Hermes），他负责：
1. OCR提取文字
2. 更新shopee-thailand-newbie-guide中的费率参数和公式
3. 通知相关人员技能已更新
