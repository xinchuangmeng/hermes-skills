# 电影级短视频文案技巧（高阶版：含《繁花》光影分镜体系）

## 一、核心原则：不是"写文案"，是"写分镜"

**每句台词都要对应一个画面设计：**
- 这句台词时，人物是什么光？
- 背景什么色调？
- 镜头是推、拉、摇、移还是固定？
- 画面里的情绪是什么？

> **普通文案：** 写的是"说什么"
> **电影级文案：** 写的是"说什么 + 观众看到什么 + 感受到什么"

## 二、《繁花》级光影体系（6种基础光位）

### 1. 伦勃朗光（主光位）

| 参数 | 说明 |
|------|------|
| **光源位置** | 人物前侧方45°，从上往下打 |
| **效果** | 脸部3/4被照亮，背光侧形成三角形光斑 |
| **情绪** | 沉稳、有力、自信、有城府 |
| **适用场景** | 讲金句、讲核心观点、讲故事高潮 |
| **AI提示词** | `Rembrandt lighting, 45-degree key light from front-side, triangle highlight on cheek` |
| **示例画面** | 人物坐在暗色背景前，一束光从右前方斜照在脸上 |

### 2. 侧光（戏剧性光位）

| 参数 | 说明 |
|------|------|
| **光源位置** | 人物正侧面90° |
| **效果** | 半边脸全亮，半边脸全暗，强烈明暗对比 |
| **情绪** | 对决、抉择、矛盾、危机感 |
| **适用场景** | 讲风险、讲陷阱、讲市场残酷面 |
| **AI提示词** | `strong side lighting from left, 90-degree, half face in shadow, dramatic contrast` |
| **示例画面** | 人物半边脸在光中，另半边完全隐入黑暗 |

### 3. 逆光（轮廓光）

| 参数 | 说明 |
|------|------|
| **光源位置** | 人物后方，对准头发/肩膀边缘 |
| **效果** | 人物边缘形成金色光晕，与暗色背景分离 |
| **情绪** | 希望、觉醒、远方、未来感 |
| **适用场景** | 讲机会、讲出路、讲未来方向 |
| **AI提示词** | `backlight, golden rim light on hair and shoulders, dark background, atmospheric` |
| **示例画面** | 人物背影或侧影，边缘镀一层金光，背景深黑 |

### 4. 蝴蝶光（女性/柔和光位）

| 参数 | 说明 |
|------|------|
| **光源位置** | 人物正前方偏上，对准鼻梁 |
| **效果** | 鼻下形成蝴蝶形阴影，面部柔和立体 |
| **情绪** | 柔软、真实、真挚、接地气 |
| **适用场景** | 讲亲身经历、讲普通人故事、讲共鸣点 |
| **AI提示词** | `butterfly lighting, soft key light from front-top, subtle nose shadow` |
| **示例画面** | 人物正对镜头，光从上方柔柔洒下，表情自然 |

### 5. 窄光（压抑/聚焦光位）

| 参数 | 说明 |
|------|------|
| **光源位置** | 人物侧后方，窄光束 |
| **效果** | 大部分脸在暗处，只有一小束光照亮局部 |
| **情绪** | 神秘、不安、压抑、思考 |
| **适用场景** | 讲行业潜规则、讲不愿面对的真相 |
| **AI提示词** | `narrow beam of light, chiaroscuro, mostly in shadow, single light source` |
| **示例画面** | 一束细光从斜后方射来，人物大半隐在阴影中 |

### 6. 全暗调/剪影（极致情绪光位）

| 参数 | 说明 |
|------|------|
| **光源位置** | 无主光，仅有背光或环境微光 |
| **效果** | 人物近乎全黑，仅轮廓可辨 |
| **情绪** | 沉重、绝望、反省、沉淀 |
| **适用场景** | 讲低谷、讲教训、讲"最坏的情况" |
| **AI提示词** | `silhouette, barely visible face, dark mood, only outline lit, noir` |
| **示例画面** | 人物剪影站在窗前，窗外有微弱城市灯光 |

## 三、《繁花》级色彩体系

| 色调 | 情绪 | 适用场景 |
|------|------|---------|
| **暗金** | 繁华、财富、野心 | 开篇定调、商业金句 |
| **深蓝/青** | 冷静、理性、思考 | 讲逻辑、拆解问题 |
| **暖橙/琥珀** | 回忆、温情、希望 | 讲故事、讲亲身经历 |
| **暗红** | 危险、警示、转折 | 讲风险、讲"但是" |
| **黑白/低饱和** | 沉重、真实、不加修饰 | 讲残酷真相、低谷时刻 |

**AI提示词示例：**
- 暗金调：`dark gold color palette, warm amber tones, vintage film look`
- 深蓝调：`deep blue color grade, cool cinematic tones, moody`
- 暖橙调：`warm orange color palette, golden hour feel, nostalgic`
- 暗红调：`dark red color grade, dangerous atmosphere, film noir`
- 黑白调：`black and white, desaturated, high contrast, grainy`

## 四、分镜格式（怎么写）

每条文案都要写成**左右对照的分镜表**：

```
【时间】  |  【画面】           |  【光线】        |  【色调】  |  【台词/独白】    |  【备注】
0:00-0:03 | 人物正面中景        | 伦勃朗光        | 暗金       | "大暑之后……"    | 语速慢，留白
0:03-0:06 | 特写人物眼睛        | 窄光            | 深蓝       | "这是规律……"    | 强调"
0:06-0:10 | 切换暗色背景剪影    | 逆光轮廓光      | 暗金+暖橙  | "泰国站……"      | 画面转亮
```

**注意：** 千万不要每句都换光线色调——那样太花哨了。一条1分钟的视频，**换3-4次就够了**。遵循"情绪驱动光线变化"的原则。

## 五、换光的时机规则

| 什么时候换光 | 怎么换 |
|-------------|--------|
| 讲一个新观点时 | 可以换光位，提示"接下来不一样了" |
| 情绪转折时 | 从暗到亮（希望），或从亮到暗（危机） |
| 关键金句时 | 用伦勃朗光或逆光，让这句有"分量" |
| 不换比换好 | 同一段情绪保持同一光位，不要频繁切换 |

## 六、完整分镜写作模板

```
🎬 【片名：XXX】 | 时长：XX秒

色彩总调：暗金复古

─── 第一幕：开场（0:00-0:08） ───
光线：伦勃朗光
色调：暗金
镜头：人物中景，缓慢推近
音效：舒缓钢琴，低频

[独白]
大暑之后，必要大寒。
这是规律。

─── 第二幕：冲突（0:08-0:20） ───
光线：侧光（切换）
色调：深蓝
镜头：侧面特写，脸部一半暗一半亮

[独白]
泰国站涨价了。
很多人慌了……
```

## 七、检查清单

一条好的电影级短视频文案：
- [ ] 每段台词指定了光位
- [ ] 每段台词指定了色调
- [ ] 全片换光3-4次，不频繁不单调
- [ ] 情绪变化驱动光线变化
- [ ] 金句配伦勃朗光或逆光
- [ ] 全片总时长控制在1-2分钟
- [ ] 开头3秒有钩子
