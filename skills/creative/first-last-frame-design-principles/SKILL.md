中英文混写效果更好，但纯英文运镜术语成功率最高。

## 一、什么是首帧和尾帧

| 概念 | 定义 | 作用 |
|------|------|------|
| **首帧**（First Frame） | 视频开场画面图片 | 告诉AI主体长啥样、场景在哪、光影方向 |
| **尾帧**（Last Frame） | 视频结束画面图片 | 告诉AI运动终点、叙事走向、变化结果 |

### 原理
- 没有参考图 → AI需同时猜"画面长啥样"和"物体怎么动"
- 有首帧 → "画面长啥样"已被定义，AI只需生成合理运动
- 有首帧+尾帧 → AI有两个确认状态做插值，生成更平滑、更有意图的运动

## 二、首帧5条铁律

### 1. 给主体留运动空间
- 别让主体填满画面，留出**10-15%边距**
- 三分法 + 运动空间：主体放在与预期运动方向相反的一侧
- **错误**：大头照填满画面 → AI不知道怎么动
- **正确**：中近景，主体一侧留空

### 2. 给隐含物理线索
- 风吹头发、重力下垂的布料 → AI知道运动方向
- 即将溅起的水花、倾斜的杯子 → AI知道接下来发生什么
- 静止物体要明确说"保持不动"

### 3. 主体特征要独特且清晰
- **错误**：灰色衬衫 + 灰色背景 → 模型无法追踪主体
- **正确**：红色夹克 + 特定发型 + 标志性配饰 → 容易跨帧追踪
- 面部光照均匀（正面光或四分之三光）
- 避免复杂图案（人字纹、细条纹、精细几何 → 产生闪烁）

### 4. 宽高比必须匹配输出
- 最终要竖屏9:16，首帧也必须是9:16
- 最终要横屏16:9，首帧也必须是16:9
- 不匹配→裁切变形或填充黑边

### 5. 分辨率至少1024×1024
- 低分辨率图 → 低质量视频，没法救
- 有瑕疵的图（变形的手、多余手指）→ 宁愿重新生成也别用
- **多花5分钟打磨首帧，省几小时重新生成**

### 常见的"好首帧"特征
- 有深度层次（前景/中景/背景）
- 运动空间充足
- 稳定元素清晰界定（不该动的区域）
- 复杂度适中（1-2个主要主体+连贯背景最佳）

## 三、尾帧设计原则

### 核心：首尾帧的差异空间 = 你讲的那个故事

| 叙事类型 | 首帧 | 尾帧 | AI自动生成 |
|---------|------|------|-----------|
| 离别 | 人站门口面对镜头 | 沿小路走远背影 | 转身→开门→走出→远去 |
| 生长 | 花苞清晨光线 | 花朵绽放午后光线 | 花瓣展开→光线转移 |
| 揭示 | 包装盒关闭状态 | 盒子打开产品可见 | 手伸向→掀开→展示 |
| 情绪变化 | 严肃表情 | 自信微笑 | 嘴角上扬→眼神变亮 |

### 尾帧4条规则

| 规则 | 说明 |
|------|------|
| **同一场景** | 不要换场景，首帧室内→尾帧室外必崩 |
| **同一个人** | 同样衣服、发型、面部特征 |
| **只改该改的** | 位置变化、表情变化、时间推移、物体状态变化 |
| **微妙变化最稳** | 小幅转头比剧烈动作成功率高10倍 |

### 什么时候不需要尾帧
- 想让AI自由发挥时（抽象/艺术/实验性内容）
- 纯氛围视频（云飘、海浪等循环）
- 首帧本身已有充分运动线索（奔跑的人、弧顶的球）
- **没有合适尾帧时宁可不用**——差的尾帧比没有更糟

## 四、品牌尾板策略（系列化账号）

### 固定+可变的组合

| 元素 | 策略 | 说明 |
|------|------|------|
| ✅ **品牌尾板**（最后3秒落版） | **每期固定** | 黑底金字logo，建立品牌识别 |
| ✅ **引导话术模板** | **每期固定句式** | "想拿XX？私信回复XX，免费发你" |
| ✅ **字幕样式** | **每期固定** | 金色大字(#FFD700)，固定字体/位置 |
| 🔄 **收尾叙事画面** | **每期不同** | 人物走远/光圈缩小/数据收敛—按当期内容设计 |
| 🔄 **尾钩预告** | **每期不同** | "下期教你用AI选爆品" |

### 品牌尾板设计模板
```
最后3秒：
画面：纯黑背景
文字：白字/金字「我和AI做跨境电商」居中
字幕：无（留白给引导话术）
音效：低频「嗡……」渐消
配音："想拿XX？私信回复XX，免费发你"
```

## 五、实战工作流速查

### 工作流1：口播转场景
- **首帧**：人物中近景，伦勃朗光，半脸阴影
- **尾帧**：同一个人缓缓转向镜头，表情变化
- **时长**：8-10秒
- **提示词**：slow head turn, natural movement, Rembrandt lighting consistent, subtle facial expression change

### 工作流2：数据展示过渡
- **首帧**：全息数据环/柱状图
- **尾帧**：某根数据柱爆发金光
- **时长**：10-12秒
- **提示词**：data bar gradually glowing from dark red to golden, particles emitting, cinematic light change

### 工作流3：情绪变化
- **首帧**：严肃表情，顶光
- **尾帧**：自信微笑，暖光
- **时长**：8秒
- **提示词**：subtle smile forming on face, warm light gradually illuminating, confident expression

### 工作流4：场景黑暗收尾
- **首帧**：人物站在光圈中
- **尾帧**：光圈缩成小光点，黑屏出现文字
- **时长**：8秒
- **提示词**：light circle shrinking from person to single point, scene fading to black, cinematic ending
