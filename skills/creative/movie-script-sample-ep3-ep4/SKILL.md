## ✅ 格式规范（以后所有脚本必须遵守）

### 输出物双重标准

每一期必须同时输出**两份**：

#### 第一份：纯文案版（用户审稿用）
```
# 第N集标题
## 口播文案
【0-8秒】文案
【8-22秒】文案
...
```
纯文案，不含画面设计，方便快速审阅内容。

#### 第二份：电影级分镜版（制作用）
每个分镜必须包含以下**6个固定字段**：
| 字段 | 必填 | 示例 |
|------|------|------|
| **画面** | ✅ | 人物从暗处转向镜头 |
| **运镜** | ✅ | 🔥 穿越镜头 / 环绕镜头 / 推镜头 / 拉镜头 / 移镜头 / 升降 / 固定 |
| **灯光** | ✅ | 🔥 伦勃朗光 / 侧光 / 顶光 / 暗金奢华 / 冷光霓虹 / 暖光 / 三点布光 |
| **配音** | ✅ | 完整文案 |
| **音效** | ✅ | 有就写，没有留空 |
| **字幕** | ✅ | 关键文字+颜色处理 |

### 每段Seedance提示词要求
```
[运镜方式] [主体] [动作] [环境] [灯光] [色调] [画质] --duration 5
```

### 灯光/运镜标记规范
- 🔥 = 核心亮点（特效最炸的灯光/运镜要标记）
- 🎥 = 运镜标记
- 💥 = 高潮/爆炸效果
