# 跨境选品可行性快速判断

## 核心原则

不是所有便宜的品都适合跨境。关键看 **总成本 vs 竞品售价** 的差距。

### ⚠️ 重要：正确解读竞品价格（最容易犯错的地方）

**不要只看一眼竞品价格就下结论，必须要做三层拆解：**

| 层 | 分析什么 | 为什么 |
|----|---------|--------|
| ① 是折扣价还是日常价？ | 页面显示的可能是促销价（-50%等），日常价可能高50-100% |
| ② 是哪个规格的价格？ | 小号引流价不赚钱，大号利润价才代表真实利润 |
| ③ 卖家是什么身份？ | 官方店/本地卖家/海外仓 = 成本结构完全不一样 |

**正确做法：**
```
看到竞品卖฿32 → 别急
先看：这是折后价吗？原价多少？→ 原价฿69，打54%
再看：这是小号还是大号？→ 小号S，大号卖฿156
最后看：谁在卖？→ Queeny官方店（本地备货）
结论：这个价格我跨境做不了，成本是人家的3倍
```

### 竞品为什么能卖这么低还赚钱

| 原因 | 拆解 |
|------|------|
| **成本端碾压** | 1688批量采购价比你低50%、货代协议运费比你低70%、有平台运费补贴 |
| **不靠这个品赚钱** | 小号引流款平本甚至亏几毛，靠大号利润款赚钱，靠全店整体盈利 |
| **隐藏收益** | 平台活动返佣、广告返点、高销量摊薄固定成本、店铺权重带来的自然流量 |

## 第一步：收集关键信息

| 信息 | 说明 |
|------|------|
| 1688采购价 | 按实际成交价，不是标价 |
| 重量 | 精确到克，越轻越好 |
| 尺寸 | 体积越小运费越低 |
| 是否一件代发 | 影响国内运费结构 |
| 目标平台/站点 | 不同平台佣金+运费不同 |

## 第二步：估算总成本

```
商品总成本(¥) = 1688采购价 + 国内运费 + 跨境运费 + 包装耗材
```

### 跨境运费估算参考

| 重量段 | 泰国(¥) | 马来(¥) |
|--------|---------|---------|
| <200g | 6-10 | 5-8 |
| 200-500g | 10-15 | 8-12 |
| 500-1000g | 15-25 | 12-20 |
| >1000g | 25+ | 20+ |

### 一件代发 vs 集运模式

| 模式 | 国内运费 | 特点 |
|------|---------|------|
| **一件代发** | ¥3-5/件 | 供应商直接发到货代，灵活但运费高 |
| **集运/囤货** | ¥0.5-2/件 | 先发国内仓再统一发跨境，量大摊薄 |

## 第三步：计算目标售价

```
净收入占比 = 1 - 佣金率 - 支付手续费 - 目标利润率
目标售价(¥) = 商品总成本 ÷ 净收入占比
目标售价(外币) = 目标售价(¥) × 汇率
```

### 各平台费率

| 平台 | 佣金 | 支付手续费 |
|------|------|-----------|
| Shopee | 5.5%-12% | 2% |
| TikTok Shop | 1%-5%(新店) | 2% |
| Lazada | 4%-6% | 2% |

> **Shopee泰国站专用精确定价公式**：已独立为 `crossborder-pricing-formula` 技能（skill-biz/crossborder-pricing-formula），含8步标准流程、SLS运费精确算法、藏价单独列出、利润验证。分析Shopee泰国站选品时，走完本技能前三步后，直接调该技能第2-7步精确计算。

### 新手建议利润率

| 阶段 | 利润率 | 说明 |
|------|--------|------|
| 起步期 | 25-30% | 有利润空间，能扛住价格战 |
| 稳定期 | 30-50% | 形成稳定出单后提升 |
| 不赚钱 | <15% | ❌ 别做 |

## 第四步：竞品价格调研

**必须上目标平台搜同类品，确认两个数：**

1. **最低价** — 市场地板价多少？这是竞品最低能卖的价格
2. **主流价** — 大部分卖家卖什么价？

### 判断标准

| 你的目标价 vs 竞品价 | 结论 |
|---------------------|------|
| 你的价 < 竞品主流价 | ✅ 有竞争力 |
| 你的价 ≈ 竞品主流价 | ⚠️ 可以，靠差异化/运营 |
| 你的价 > 竞品主流价30%+ | ❌ 难卖，除非有独特价值 |
| 你的成本 > 竞品最低价 | ❌ **成本倒挂，别做** |

> **成本倒挂** = 你的进货+运费成本比竞品的售价还贵 → 怎么做都是亏

## 第五步：最终决策

```
                  ┌─────────────────┐
                  │ 找到1688一个品    │
                  └────────┬────────┘
                           ↓
                  ┌─────────────────┐
                  │ 重量 < 500g？    │ ← ❌ 太重，跨境运费扛不住
                  └────────┬────────┘
                           ↓ 是
                  ┌─────────────────┐
                  │ 成本 < 竞品价70%？│ ← ❌ 成本倒挂，别碰
                  └────────┬────────┘
                           ↓ 是
                  ┌─────────────────┐
                  │ 利润 > 25%？     │ ← ❌ 太薄，没安全垫
                  └────────┬────────┘
                           ↓ 是
                  ┌─────────────────┐
                  │ ✅ 可以做！      │
                  └─────────────────┘
```

## 6️⃣ 踩坑记录

| 坑 | 怎么避免 |
|----|---------|
| 只看1688价便宜就冲 | 必须算上所有运费再判断 |
| 低估重量 | 一定要问供应商精确重量，别猜 |
| 只看淘宝价不看Shopee | 不同平台价格体系完全不同 |
| 一上来就囤货 | 新手一律一件代发先测品 |
| 忽略包装 | 跨境运输损耗大，包装要结实 |

## 8️⃣ 特别注意：服务器限制

> 1688和Shopee等电商网站有反爬机制，服务器IP可能被拦截或跳验证码。
> 1688返回阿里云盾验证页、Shopee是客户端渲染（HTML无商品数据）。
> **不要在服务器上强行抓取，让用户直接在App/网页查看更高效。**
