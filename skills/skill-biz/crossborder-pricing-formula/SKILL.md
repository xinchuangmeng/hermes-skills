## 二、关于"藏价"机制（来自虾皮大学官方定义）

> 数据来源：虾皮大学「物流规则 → 跨境物流成本（藏价）与物流运费一览」2025-11更新

**藏的到底是什么？**
```
卖家支付运费 = 跨境物流成本（藏价） + 买家支付运费
```

- **跨境物流成本（藏价）** = 卖家实际承担的物流成本 → **必须加入商品价格**
- **买家支付运费** = 买家看到的运费（平台系统显示）

**关键原则（来自官方）：**
- 藏价≠包邮！不管包不包邮，藏价都要藏进去
- 如果要包邮，使用【运费补贴】功能，不是在售价里取消藏价
- 多件出单场景可以分摊藏价（例如：85%的订单是2-3件，可以只藏2/3的藏价）

## 三、核心公式体系

### 公式速查表

| 场景 | 公式 | 说明 |
|:----|:----|:------|
| **免佣期（前3个月）** | 售价(¥) = 总成本 ÷ (0.98 - 利润率) | 只扣2%交易手续费 |
| **正常期不活动** | 售价(¥) = 总成本 ÷ (0.7937 - 利润率) | 扣20.63% |
| **正常期+活动** | 售价(¥) = 总成本 ÷ (0.7437 - 利润率) | 扣25.63% |
| **泰铢换算** | 售价(฿) = 售价(¥) × 5 | |
| **买家含税总价** | 含税价(฿) = 售价(฿) × 1.07 | 跟竞品比价时用这个 |
| **利润验证** | 利润 = 售价(¥) - 进价 - 国内运费 - 藏价 - 平台扣费 | |

### 两步定价法（新手推荐）

**第1步：用正常期（不活动）费率定原价（锚点价）**
```
原价(¥) = 总成本 ÷ (1 - 0.2063 - 目标利润率)
原价(฿) = 原价(¥) × 5 → 凑整
```

**第2步：设折扣后，用免佣期费率算实际到手**
```
真实售价(฿) = 原价(฿) × 折扣率（通常65-75折）
真实售价(¥) = 真实售价(฿) ÷ 5
平台扣费(¥) = 真实售价(¥) × 2%（免佣期只扣交易费）
到手(¥) = 真实售价(¥) - 平台扣费
利润(¥) = 到手 - 总成本
利润率 = 利润 ÷ 真实售价（验证是否达标）
```

**第3步：含VAT竞价检查**
```
买家含税总价(฿) = 真实售价(฿) × 1.07
→ 用这个数跟竞品比价，确保有竞争力
```

> 关键逻辑：用正常期费率定原价（高标价）→打折扣出真实售价→用免佣期费率算利润
> 3个月后直接取消折扣，原价就是正常价，无缝衔接

## 四、完整计算示例

### 已知：600g收纳盒，1688进价¥4.94，国内运费¥3，免佣期，不参加免运活动

**第1步：计算藏价（SLS运费部分）**
```
SLS运费 = (600÷10-1) × 0.25 = ¥14.75（此为预估费率）
藏价 = SLS运费 - 买家承担运费 = 需查运费计算器确认准确值
```
> 实际藏价需使用Shopee运费计算器获取最新SLS费率

**第2步：总成本**
```
总成本 = 进价¥4.94 + 国内¥3 + 藏价 = 待藏价确认后计算
```

**第3步：按两步定价法计算**
```
原价(¥) = 总成本 ÷ (1 - 0.2063 - 0.25)
原价(฿) = 原价(¥) × 5
设7折 → 真实售价(฿)
买家含税价(฿) = 真实售价(฿) × 1.07
```

**利润验证：**
```
真实售价(¥) = 真实售价(฿) ÷ 5
扣费(¥) = 真实售价(¥) × 2%
到手(¥) = 真实售价(¥) - 扣费
利润(¥) = 到手 - 总成本
利润率 = 利润 ÷ 真实售价 → 应≥25%
```

## 六、强制约束（新手专用）

1. ✅ 必须先确认当前是**免佣期还是正常期**
2. ✅ 必须先确认**是否参加FSS/CCB活动**（影响总费率）
3. ✅ 藏价必须单独列出，**不可省略不可混入总成本不提**
4. ✅ VAT 7%买家承担，但必须计入**含税总价**跟竞品比
5. ✅ 最后必须给凑整建议价 + 含税竞价比价

## 七、常见坑位

| 坑 | 后果 | 避免方法 |
|:---|:----|:---------|
| ❌ 用旧费率8% | 定价低了利润亏 | 现在官方18.63%，更新了 |
| ❌ 不算藏价 | 运费吃掉利润 | 藏价必须单独算入售价 |
| ❌ 不查运费计算器 | 运费估不准 | 去企业微信Shopee扫码下载 |
| ❌ 不打折直接标 | 竞品都打折标的比你低 | 用两步定价法标高价再打折 |
| ❌ 不算VAT竞价 | 含税价比竞品贵 | ×1.07后跟竞品比 |
| ❌ 免佣期用正常期费率定价 | 定太高卖不动 | 两步定价法：正常期定锚→免佣期算到手 |
