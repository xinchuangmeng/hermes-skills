# 腾讯云服务器选型指南

## 概述

腾讯云主要提供三类服务器产品：轻量应用服务器（Lighthouse）、云服务器CVM、GPU云服务器，外加无服务器的云函数SCF。选型核心：根据场景选类型，根据预算选配置。

---

## 一、产品全景对比

| 维度 | 轻量应用服务器 | 云服务器CVM | GPU云服务器 | 云函数SCF |
|------|--------------|------------|------------|----------|
| **定位** | 傻瓜式、便宜 | 专业级、灵活 | 带显卡的高性能计算 | 无服务器、按需执行 |
| **操作难度** | ⭐ 极简 | ⭐⭐⭐ 复杂 | ⭐⭐⭐ 复杂 | ⭐⭐ 中等 |
| **价格模式** | 套餐年付（几十~几千/年） | 选配月付/按量 | 月付/按小时 | 按调用次数+运行时长 |
| **显卡** | ❌ 不支持 | 可选挂载 | ✅ 标配T4/V100/A100 | ❌ 不支持 |
| **灵活性** | 低（只能整体升降配） | 高（CPU/内存/硬盘/带宽独立选） | 高 | 只写代码不管服务器 |
| **适合谁** | 个人/小项目/新手 | 企业/复杂架构 | AI/深度学习/渲染 | 定时任务/事件触发 |

## 二、轻量应用服务器（当前在用）

### 适用场景
- ✅ 跑Hermes Agent
- ✅ 搭建网站/博客
- ✅ 后端API服务器（小项目）
- ✅ 开发测试环境
- ✅ 跨境电商独立站
- ⚠️ 跑Docker（2G内存偏紧，4G以上可以）
- ❌ 跑大模型推理
- ❌ 视频批量处理

### 2026年价格参考

| 配置 | 价格 | 推荐度 |
|------|------|--------|
| 2核2G / 3M / 40G / 200G流量 | 38-68元/年 | ⭐ 最便宜 |
| **2核2G / 4M / 50G / 300G流量** | **99元/年** | ⭐ **当前在用** |
| 2核4G / 5M / 60G / 500G流量 | 188元/年 | ⭐ 性价比高 |
| 4核4G / 3M / 40G / 300G流量 | 38-99元/年（秒杀） | ⭐ 适合Docker |
| 4核8G / 12M / 180G / 2000G流量 | 880元/15个月 | ⭐ 性能款 |
| 4核16G / 14M / 220G / 2500G流量 | 99元/月 | ⭐ 高配 |

### 优缺点
**优点：** 便宜、大带宽、SSD硬盘、开箱即用、一键部署应用
**缺点：** 不能加显卡、限制月流量、不能单独升级部件、不能搭建集群

## 三、云服务器CVM

### 适用场景
- 高并发网站/App
- 复杂分布式系统
- 需要自建VPC网络
- 需要弹性伸缩的业务
- 可挂载GPU显卡（选对应机型）

### 与轻量对比
| 对比项 | 轻量 | CVM |
|--------|------|-----|
| 网络配置 | 自动，无需操心 | 自己配VPC/安全组/子网 |
| 计费 | 套餐价，年付为主 | 选配计费，支持按量/包年/竞价 |
| 价格 | 几十~几百/年 | 几十~几百/月 |
| 弹性IP | 不支持 | 支持绑定弹性IP |
| 内网互联 | 有限制 | 支持VPC、负载均衡、云数据库内网连接 |

## 四、GPU云服务器（带显卡）

### 显卡选型

| 显卡型号 | 级别 | 适合场景 | 参考月价 | 按小时价约 |
|---------|------|---------|---------|-----------|
| **T4** | 入门级 | 轻量AI推理、视频转码、轻量生图 | 120元/月 | ~0.3元/小时 |
| **V100** | 中高端 | 深度学习训练、大模型推理、生图 | 220-240元/月 | ~0.6元/小时 |
| **A100** | 旗舰级 | 大模型训练、科学计算 | 1888元/15天 | ~5元/小时 |

### 使用模式
1. **包年包月**：长期使用，价格优惠
2. **按量计费**：按小时租，用完即关，适合间歇性任务
3. **竞价实例**：最便宜，但可能被系统回收

### 可做的事情
- 部署大模型（DeepSeek、Qwen等本地推理）
- AI绘画（Stable Diffusion）
- AI视频生成
- **数字人直播**（实时渲染+推流，需要V100或以上）
- 深度学习训练

## 五、云函数SCF（无服务器）

### 核心特点
- 不买服务器，只写代码
- 按实际运行时间计费（没调用不收钱）
- 自动弹性伸缩
- 支持Python/Node.js/Java/Go等

### 适用场景
- 定时任务（cron替代）
- 图片/视频上传后自动处理
- Webhook接收处理
- 轻量API接口
- 数据爬虫

### 不适用场景
- 需要长连接的服务器（如Hermes）
- 大内存/GPU计算
- 复杂后端系统

## 六、地域选择

| 地域 | 延迟 | 需要备案 | 适合用户 |
|------|------|---------|---------|
| 广州/上海/北京 | 20-50ms | ✅ 必须备案 | 面向中国大陆用户 |
| 中国香港 | 40-80ms | ❌ 不需要 | 不想备案、兼顾国内+东南亚 |
| 新加坡/东京 | 80-150ms | ❌ 不需要 | 面向海外/跨境电商 |
| 欧美 | 150-300ms | ❌ 不需要 | 面向欧美用户 |

**选择原则：** 用户距离服务器越近，速度越快。
- 南方用户 → 广州
- 北方用户 → 北京
- 面向全国 → 上海
- 不想备案 → 香港
- 东南亚业务 → 新加坡

## 七、选型决策树

```
你要做什么？
├── 跑Hermes/脚本/小网站
│   └── 轻量服务器（2核2G ~ 4核8G）
├── 需要Docker跑多个服务
│   └── 轻量服务器（4核8G起步）或 CVM
├── 需要GPU显卡
│   ├── 数字人直播 → GPU服务器 V100+
│   ├── 大模型推理/生图 → GPU服务器 T4/V100
│   └── 大模型训练 → GPU服务器 A100
├── 企业级/高并发/复杂架构
│   └── CVM（按需选配）
├── 定时任务/事件触发
│   └── 云函数SCF
└── 面向海外/不想备案
    └── 香港或海外节点
```
