# Hermes Agent Computer Use 功能指南

## 功能说明

Computer Use 让 Hermes Agent 能像人一样操控电脑鼠标键盘：
1. 收到指令 → 截图屏幕 → 视觉模型分析 → 确定点击坐标 → 移动鼠标操作 → 验证结果

## 安装与配置

```bash
# 1. 更新到最新版（v0.13+，该功能在v0.12被回滚，需开发版或更新版）
hermes update

# 2. 安装CUAdriver
hermes computer-use install

# 3. 系统权限设置（Mac）
# 系统设置 → 安全与隐私 → 辅助功能 → 勾选CUADriver + Terminal
# 系统设置 → 安全与隐私 → 录屏权限 → 勾选CUADriver + Terminal
# Windows类似：需要管理员权限+辅助功能授权

# 4. 运行
hermes computer-use
# 或直接hermes chat中用视觉模型
```

## 模型要求

| 要求 | 说明 |
|------|------|
| **必须是视觉模型** | 需要能看懂截图，DeepSeek不支持 |
| **推荐模型** | Qwen 3.5/3.6-Plus（视觉版）、GPT-4o、Claude Vision |
| **性能** | 模型越强，点击越准，纠错能力越好 |

## 适用场景

| 场景 | 描述 | 可行性 |
|------|------|--------|
| 自动上架商品 | 打开Shopee后台 → 填信息 → 上架 | ✅ 可行 |
| 1688搜品比价 | 打开1688 → 搜关键词 → 对比价格 | ✅ 可行 |
| 桌面软件操作 | 打开剪映 → 导入素材 → 自动剪辑 | ⚠️ 较慢，偶尔点错需纠正 |
| 批量处理订单 | 打开卖家中心 → 处理订单 | ✅ 可行 |
| 服务器端操作 | 我（Hermes服务器版）不能跑Computer Use | ❌ 需在本地电脑运行 |

## ⚡ 版本升级实战经验（2026-05-14更新）

### 当前版本的升级陷阱

**场景：** 服务器上的Hermes Agent落后3880个commit（v0.13.0 → 最新版）。
**问题：** `git pull --ff-only` 反复失败。

### 失败原因

| 原因 | 说明 |
|------|------|
| **Untracked文件冲突** | 用户手动复制的中文汉化文件（README.zh-CN.md, locales/zh.yaml, agent/*.py等）被git视为新文件，阻止merge |
| **Optional-skills目录冲突** | 之前存在的可选技能目录与新版本的文件结构不一致 |
| **带宽限制** | 3880个commit的差异量太大，下载超时（单次pull耗时>60s） |

### 正确升级策略

**不要直接在服务器上跑 `hermes update` 或 `git pull`，而是：**

```
# 如果只是小版本更新
hermes update    # Hermes内置更新命令，处理依赖

# 如果落后太多（1000+ commits）
# 推荐：新机器上全新安装，而不是升级
pipx install hermes    # 全新安装更快更干净
```

### 架构约束：服务器 vs 本地电脑

| 部署位置 | 版本策略 | Computer Use |
|---------|---------|-------------|
| **腾讯云服务器（我）** | 保持v0.13.0，够用不动 | ❌ 不能跑，需要本地GUI |
| **你本地电脑（龙虾）** | 全新安装最新版 | ✅ 可以跑，装CUAdriver+授权 |

### 关键认知

> **不要在我（服务器版Hermes）上升级Computer Use。** 这个功能需要在你的本地电脑上跑，因为需要操控鼠标键盘。服务器上没显示器，操作不了桌面软件。

> **升级策略：** 如果当前版本够用（v0.13.0有核心功能），不要为了一个功能花大量时间升级。等有明确需求了再在新机器上全新安装。

## 注意事项

| 事项 | 说明 |
|------|------|
| **当前状态** | v0.12.0中该功能已回滚，需要更新到开发版或后续正式版 |
| **运行位置** | 必须在你自己的电脑上跑（不是服务器） |
| **速度** | 比人操作慢，每次操作需要截图→分析→执行→验证循环 |
| **可靠性** | 点错时会自动纠正（如文章演示的点错菜单→重新点击） |
| **前置条件** | 需要本地已安装Hermes Agent + 已配视觉模型 |
