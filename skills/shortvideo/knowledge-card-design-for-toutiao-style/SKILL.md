# 知识卡片设计流程（针对抖音图文短视频）

## ⚠️ 核心原则

**Pillow/Python纯文字卡片（白字黑底、无设计）会被用户拒绝。** 但**在有明确参考风格+设计模板的情况下，Pillow生成的卡片是可接受的。**

选择标准：
1. **风格①（米黄笔记风/极简风）** → 用Pillow做 ✅ 纯文字+单色底，服务器能做好
2. **风格②（警示牌风/插画风）** → 用Pillow做底层背景 ✅ + 参考代码见本技能末尾附录
3. **风格③（暗黑特效风/电影风）** → 不能代码做 ❌ 需要剪映特效层叠加
3. **风格③（暗黑特效风/电影风）** → 不能代码做 ❌ 需要剪映特效层叠加

**最佳流程：服务器出图 → 敬哥导入剪映图文成片 → 可加字幕特效**

正确姿势：**出设计方案（文案+配色+排版指示），让用户用Canva/剪映/即梦AI做。**

## 🔑 关键经验（2026/5/11更新）

### 让用户传视频参考的流程
抖音/IP被封时（服务器IP被抖音拉黑403）：
1. 首先试用飞书直接传文件（最快）——`+ → 文件 → 选择MP4`
2. 文件>200MB走scp传服务器（给用户完整命令含路径）—— `scp "C:\path\to\file.mp4" agentuser@42.193.201.6:/home/agentuser/video_analysis/`
3. scp密码不对→开HTTP上传页面备用
4. 收到后ffmpeg抽帧→Vision分析→确定风格方向
5. 如果连scp也传不了，用HTTP上传页面（启动Python http.server）

### 分析用户视频找风格参考流程
```bash

ffmpeg -i input.mp4 -ss 00:00:0$i -frames:v 1 -update 1 output_frame.jpg -y

# 2. 用vision_analyze分析每帧卡片风格

# 3. 对比分析（多个视频同时分析）
# 让用户提供"简单版"和"困难版"两种风格参考
# 输出对比表：风格名 | 背景 | 文字排版 | 难度 | 制作方式 | 适不适合我们
```

### 用户偏好的工作模式（重要）

- 分工应该：
  - **Hermes小书童**：文案产出、风格分析、卡片设计方案、视频分析、发任务到Hermes之家群

## 一、设计前的准备

### 参考对标
每次设计前，先让用户分享一个抖音上的图文短视频参考链接。参照：
- 整体排版风格（字间距、行距、对齐方式）
- 配色方案（渐变色、纯色、图片背景）
- 信息密度（一张卡放几个信息点）
- 视觉元素（图标、装饰线、表情符号）

### 每期6-7张卡（60秒内）
| 编号 | 类型 | 分配 |
|:----:|:----|:----|
| 1 | **封面** | 悬念标题+品牌标识 |
| 2 | **痛点引入** | 问题/场景代入 |
| 3-5 | **核心干货** | 每个要点1张卡(编号①②③) |
| 6 | **结尾** | 预告+品牌+引导关注 |

## 二、卡片设计方案模板

### 每张卡必须写清楚

```markdown
### 卡片N：卡片名
| 元素 | 设计 |
|------|------|
| **背景** | 颜色/渐变/图片底 |
| **标题** | 文字内容+字号+颜色+位置 |
| **正文** | 每个要点分行写，注意间距 |
| **装饰** | 图标/emoji/分隔线/编号 |
| **风格** | 一句话描述整体感觉 |
```

### 设计要点
1. **每张卡只讲1个概念**，不堆砌
2. **字号要大**（手机能看清，标题建议60-72px，正文36-50px）
3. **颜色对比鲜明**（深底浅字或浅底深字）
4. **加图标/箭头/数字**等视觉元素（圆形编号效果最好）
5. **留白充足**，字不要塞太满（上下左右至少40px）
6. **统一调性**——6张卡保证风格一致（同色系、同字体）
7. **参考先行**——让用户先分享抖音上的参考视频再动手做，避免方向错误

## 四、制作方式推荐（含服务器方案）

### 用户手动制作
| 工具 | 推荐度 | 原因 |
|:----|:------:|:-----|
| **Canva** | ⭐⭐⭐⭐⭐ | 大量免费模板，拖拽操作，一键统一风格 |
| **即梦AI** | ⭐⭐⭐⭐ | 文生图能做背景，但文字排版要手动调 |
| **PPT/Keynote** | ⭐⭐⭐ | 能导出图，但效率低 |
| **剪映图文成片** | ⭐⭐⭐ | 只能简单调整，不适合精细设计 |

**首推Canva**——搜「知识卡片」「电商干货」关键词，找到合适的模板直接套。

### 服务器自动出图（Pillow方案）

适用于风格①（极简笔记风）和风格②（警示牌风），当用户没时间自己做卡时：

**风格①：米黄笔记风**
- 背景：米黄色/奶油色纯底（#F5F0E0或#FFF8DC）
- 文字：深灰/黑色无衬线字体
- 排版：左对齐列表，大标题居中
- 装饰：编号（①/②/③）、emoji图标
- 示例参数：字体72px(标题)/50px(正文)，间距120px

**风格②：警示牌风**
- 背景：深红色纯底（#C62828），顶部深红警示条（#B71C1C）
- 文字：白色粗体，居中对齐大标题+左对齐列表
- 装饰：黄色圆形编号圈（#FFCA28）、底部红白斜条纹警戒带
- 核心要素：每张卡用Pillow绘制背景层 + 从AI生图工具取插画人物合成
- AI生图提示词参考："扁平卡通插画风格，一个沮丧/焦虑表情的年轻男子举着手机，简洁几何形，纯色背景，无复杂阴影"

**剪映完善：** Pillow出图后，告诉用户在剪映加上文字动画、入场特效、BGM即可。

## 五、剪映图文成片操作

1. 在Canva做好6张卡片图并导出为PNG
2. 打开剪映 → 「图文成片」
3. 一次性导入6张图（按顺序）
4. 选BGM（轻知识类，音量可调大，无配音）
5. 每张图调为8-10秒 → 总时长55-60秒
6. 检查字幕识别是否正确
7. 导出：竖屏9:16，1080p

## 附录：Pillow生成风格②（警示牌风）卡片参考代码

```python
from PIL import Image, ImageDraw, ImageFont

W, H = 1080, 1920
img = Image.new('RGB', (W, H), (20, 20, 20))
draw = ImageDraw.Draw(img)

# 卡片主体：红色底
card_w, card_h = 900, 1100
card_x = (W - card_w) // 2
card_y = (H - card_h) // 2 - 100
draw.rectangle([card_x, card_y, card_x+card_w, card_y+card_h], fill=(200, 45, 45))

# 顶部警示条（深红色）
strip_h = 100
draw.rectangle([card_x, card_y, card_x+card_w, card_y+strip_h], fill=(180, 30, 30))

# 底部红白斜条纹警戒带
stripe_h = 80
stripe_y = card_y + card_h - stripe_h
for i in range(0, card_w, 40):
    points = [
        (card_x + i, stripe_y),
        (card_x + i + 50, stripe_y),
        (card_x + i + 40, stripe_y + stripe_h),
        (card_x + i - 10, stripe_y + stripe_h)
    ]
    draw.polygon(points, fill=(255, 255, 255) if (i//40)%2==0 else (200,45,45))

# 字体加载（优先中文字体）
try:
    font_big = ImageFont.truetype("wqy-zenhei.ttc"或"DejaVuSans.ttf", 72)
    font_mid = ImageFont.truetype(..., 50)
    font_small = ImageFont.truetype(..., 36)
except:
    font_big = ImageFont.load_default()

# 文字绘制示例
draw.text((card_x+60, card_y+20), "⚠️ 敏感内容", fill=(255,255,220), font=font_mid)

# 黄色圆形编号
for num, text in [("01", "问题1"), ("02", "问题2"), ("03", "问题3")]:
    circle_r = 40
    cx = card_x + 120
    draw.ellipse([cx-circle_r, qy, cx+circle_r, qy+circle_r*2], fill=(255,200,50))
    # 白色文字在编号旁
    draw.text((cx + 80, qy + 20), text, fill=(255,255,255), font=font_mid)
    qy += 120
```
