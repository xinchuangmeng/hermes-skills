# 参考视频卡片风格分析流水线

## 适用场景
用户有抖音/视频号的参考录屏视频（MP4），想让你分析其中的卡片设计风格并复现。

## 三步流程

### 第一步：提取关键帧

用户上传MP4到服务器后，用ffmpeg按时间间隔提取关键帧：

```bash
# 起始帧
ffmpeg -i input.mp4 -ss 00:00:00 -frames:v 1 -update 1 /path/frame_0.jpg -y 2>/dev/null

# 按秒提取多帧
for i in 0 2 4 6 8; do
  ffmpeg -i input.mp4 -ss 00:00:0$i -frames:v 1 -update 1 /path/frame_$i.jpg -y 2>/dev/null
done
```

**注意事项：**
- 大文件（>100MB）用 `-ss` 快速定位，不要从头解码
- 强制用 `-update 1` 才生成单图，不加会出现奇怪的错误
- 视频过长的，每2-3秒抽一帧

### 第二步：Vision AI风格分析

用 `vision_analyze` 工具分析每张关键帧，重点提取：

| 分析维度 | 具体内容 |
|---------|---------|
| 背景颜色 | 纯色/渐变/纹理？具体色值？ |
| 文字排版 | 对齐方式、字号层级、字体风格 |
| 装饰元素 | 图标、边框、编号、警戒线等 |
| 整体风格 | 简约笔记风/警示牌风/暗黑科技风/其他 |
| 制作难度 | 易/中/难，什么工具能实现 |

**分析提示词模板：**
> "分析这张卡片的完整设计风格：背景颜色/图案、文字排版（字体、大小、位置、间距）、装饰元素、整体风格类型"

### 第三步：产出卡片方案

根据分析结果输出方案，包含：
1. 每张卡片的具体文案
2. 统一的设计规范（颜色、字体、装饰）
3. ASCII示意图展示布局
4. 制作指引

## 常见风格对照表

| 风格 | 背景 | 文字 | 装饰 | 难度 | 实现方式 |
|------|------|------|------|:----:|---------|
| 米黄笔记风 | 米黄/奶油纯色 | 黑色无衬线左对齐 | 编号/分隔线 | ⭐ | Pillow/Python |
| 警示牌风 | 深红底 | 白色粗体+黄色编号圈 | 斜条纹警戒带/警示标签 | ⭐⭐ | Pillow/Python+AI生图 |
| 暗黑科技风 | 纯黑+粒子/光束 | 金色发光立体字 | 星光粒子/丁达尔光 | ⭐⭐⭐⭐ | 剪映特效（服务器做不了） |

## 经验教训
- 用户上传大文件时先看文件大小，scp命令一次全传更快
- 飞书文件链接需登录才能下载，服务器无法直接访问，让用户用scp
- Vision AI分析时一次分析一张最准确，批量分析会丢失细节
- 卡片设计风格分析完成后，先给用户看ASCII示意图确认方向，再出完整方案
