# 短视频AI素材智能采集剪辑通用技能

## 一、前置素材标准化强制执行规则

1. 所有剪辑素材统一规格：1080P、25帧、H.264编码
2. 拒绝模糊、竖屏裁切不全素材
3. 自动预处理：剔除抖动、过曝、黑屏、无意义空镜废片
4. 音频自动处理：全局降噪、音量归一化，标记人声停顿、金句段落
5. 素材自动按「开场/产品特写/口播金句/结尾引导」分类打标归档

## 二、通用剪辑固定规则

### 镜头时长标准
- 抖音快剪：0.8-1.5秒/镜
- 常规口播：1-2秒/镜
- 剧情慢节奏：2-3秒/镜

### 转场规范
- 优先硬切、淡入淡出
- 禁用花哨特效
- 每3-4个镜头仅使用1次转场

### 人声保留规则
- 静音阈值固定1.2秒
- 保留自然换气停顿
- 严禁截断关键台词

### 字幕规范
- 默认黑体、标准字号
- 金句关键字自动放大高亮
- 收录行业专有名词自建词库
- 杜绝错别字与字幕时间错位

## 三、分场景剪辑调用模板

### 模板一：15秒带货口播
**固定结构：** 3秒开场抓人 → 中间产品多镜头轮播 → 2秒金句定格 → 结尾引导停留
- 镜头跟随BGM卡点快切
- 关键产品镜头强制锁定保留，不自动删减

### 模板二：30秒知识口播
**固定结构：** 慢镜头节奏，分句精准匹配对应画面
- 不盲目快切，优先语义匹配
- 完整保留整句台词，不掐头去尾

### 模板三：好物混剪
**固定结构：** 多素材随机轮播，自动过滤相似重复镜头
- 跟随BGM鼓点自动切镜
- 统一画面色调滤镜，保持风格一致

## 四、模型固定配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 创作随机性 | 0.1-0.3 | 保持剪辑风格稳定不杂乱 |
| 字幕识别 | Whisper-large | 提升识别精准度 |
| 画面分析 | ViT光流模式 | 精准识别人脸/产品/场景 |
| 关键帧 | 手动标记必留镜头 | 智能体禁止擅自删除 |

## 五、剪辑效果差快速排障规则

| 问题 | 解决 |
|------|------|
| 节奏混乱 | 锁定固定镜头时长、减少转场频率 |
| 关键台词被剪 | 调高静音阈值、标记金句锚点 |
| 画面重复 | 开启相似镜头过滤、轮换调用模板 |
| 字幕错乱 | 导入专有名词词库、微调字幕时间轴±0.1秒 |

## 六、智能素材全网采集执行流程

1. 接收剪辑需求 → 自动拆解提取（场景/风格/时长/分辨率/可商用/核心关键词）
2. 免费无版权渠道依次检索：Pexels、Pixabay、Mixkit、Videvo
   - 搜索句式：`关键词 + 1080P + 5至15秒 + 横版 + 无水印 + 免费商用`
3. 免费素材不足 → 切换国内正版商用渠道：光厂、制片帮
   - 搜索句式：`中文关键词 + 电商带货 + 1080P + 正版商用`
4. 自动过滤：带水印、竖版、画质模糊、黑边、时长不符、无效素材
5. 下载后自动统一标准化：1080P、25fps、H.264、横版制式
6. 按场景、时长、风格、商用属性自动打标分类入库
7. 最终输出5-10个最优素材链接，附带标签备注供直接调用

## 七、智能体自学习记忆规则

1. 每次剪辑完成 → 自动记录人工标注（保留片段、删减片段、节奏调整点）
2. Hermes与龙虾自动沉淀用户剪辑审美偏好
3. 累计10次反馈 → 固化专属个人剪辑风格
4. 每周自动复盘剪辑问题短板
5. 迭代更新剪辑模板与采集搜索规则
6. 越使用越贴合个人创作习惯
